# Codex Rule Review

250건 샘플에서 확인한 핵심은 다음과 같습니다.

| type | count | 판단 |
|---|---:|---|
| parts | 72 | `본체/유닛` 단독 키워드는 너무 넓고, `본체만/유닛만/한쪽/낱개/단품/좌우 유닛` 문맥이 고정밀 |
| accessory | 47 | 스트랩/밴드/케이스/거치대 계열은 유지 |
| damaged | 47 | 기존 `하자` 단독 룰은 오탐이 큼. `하자 없음/무하자`는 제외해야 함 |
| counterfeit | 8 | `타오바오/타오바이/짭/가품/레플리카`는 즉시 제외급 |
| deep_discount | 5 | AirPods Max 8만원 이하 같은 매물은 Tier2 확인 또는 제외 후보 |

## 반영한 코드 변경

- `CALLOUT_KEYWORDS`에 가품/레플리카/타오바오 표현 추가
- `partsHits()` 추가: 본체/유닛 단독 단어 대신 단품/만/좌우 유닛 문맥으로 판정
- `damagedHits()` 추가: `하자 없음`, `하자 전혀 없음`, `무하자`는 damaged 오탐에서 제외
- UI에서는 가품/레플리카/타오바오 의심 매물을 `제외` 뱃지로 처리하고 기본 전체 목록에서 숨김
