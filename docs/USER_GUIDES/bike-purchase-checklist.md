# 자전거 매물 추천 받기 전 체크리스트 (미뇨이)

> Wave 93 적용. 미뇨이가 자전거 매물 추천을 시작합니다. 안전 거래를 위해 추천 카드 클릭 전 반드시 다음을 확인하세요.

## 1. 프레임 사이즈 — 본인 체형에 맞는지

자전거는 사이즈가 안 맞으면 못 타거나 부상 위험이 있습니다.

### 로드 자전거 (TCR / Defy / Tarmac / Emonda / Allez / Domane 등)

| 키 (cm) | 프레임 사이즈 |
|---|---|
| 155~165 | 49~52cm (XS/S) |
| 165~175 | 52~54cm (S/M) |
| 175~185 | 54~56cm (M/L) |
| 185~195 | 56~58cm (L/XL) |

### MTB (Big Nine / Marlin / Talon / XTC 등 29인치)

| 키 (cm) | 프레임 사이즈 |
|---|---|
| 155~165 | XS/S |
| 165~175 | M |
| 175~185 | L |
| 185~195 | XL |

→ **매물 제목 또는 본문에 프레임 사이즈 명시 없으면 셀러에게 직접 확인**.

## 2. 사고/크랙 이력

탄소 프레임 (Carbon)은 한번 사고나면 외부에 잘 안 보여도 내부 구조 손상 → 갑자기 부러질 위험. 알루미늄은 사고 후 도색만 새로 한 매물 多.

### 확인 사항

1. **셀러에게 직접 질문**: "사고 이력 있나요? 무사고인가요?"
2. **현장 점검**: 시트튜브, 탑튜브, 헤드튜브, BB(바텀브라켓) 주변 크랙/도색 새로 한 흔적 확인
3. **시리얼 번호 확인** (BB 아래 각인) — 도난 자전거 확인 (특히 Brompton)
4. **영수증/구매 증빙** — 도난 위험 ↑ 브랜드는 필수

### 자전거 도난 risk 순위

1. **Brompton** (한국 도난 1순위)
2. **Pinarello / Cervelo** (200만+ 모델)
3. **카본 로드** (Specialized Tarmac / Trek Emonda / Giant TCR Advanced 등)

## 3. 주행거리 확인

- 카본 로드: 1만 km 이상은 프레임 수명 검토 필요
- 알루미늄 로드/MTB: 5천 km 이상 사용감 점검
- 폴딩/하이브리드: 출퇴근용 1~3년 사용 보통

→ 셀러에게 "주행거리 어느 정도 되시나요?" 질문 권장.

## 4. 부품 교체 여부

순정 부품 vs 업그레이드 매물 시세 다름.
- "변속기 교체", "휠셋 교체" 매물은 가격 + α 정당화 가능
- 단 "구성품 빠짐", "핸들/안장만 분리 판매" 매물은 본품 아님 (미뇨이 reject)

## 5. 직거래 시 안전 가이드

- 가능하면 자전거 매물은 **직거래 + 현장 시승** 권장
- 거래 전 본인 체형/사이즈 측정
- 셀러 본인 신원 확인 (도난 자전거 차단)
- 가격 시세 대비 너무 싼 매물 (-50% 이상)은 의심
