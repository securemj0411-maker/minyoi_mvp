# 2026-06-05 Wave 1151 — 당근 피드 끝 안내를 희소성/알림 CTA로 정리

## 결정
- 당근/가까운순 피드 끝 안내는 기능 설명보다 사용자의 의문에 답하는 방향으로 바꾼다.
- `이게 끝인가?`에 대한 답으로, 내 근처에서 수익성·상태·판매 속도를 통과한 매물이 극소수라는 점을 설명한다.
- 새로 올라오는 기회는 `새매물 알림 받기` CTA로 자연스럽게 이어지게 한다.

## 구현
- `src/components/explore-client.tsx`
  - 당근 focused view의 끝 카드 제목을 `내 근처에서 수익 나는 매물은 극소수예요`로 변경했다.
  - 설명을 `차익, 상태, 판매 속도를 통과한 결과`와 `새매물 알림` 연결 문구로 변경했다.

## 보류
- 지역별 후보 수/커버리지 수치를 카드에 직접 보여주는 작업은 별도 데이터 응답 확장이 필요해 이번 범위에서 제외했다.
