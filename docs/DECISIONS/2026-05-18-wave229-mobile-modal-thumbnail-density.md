# 2026-05-18 Wave 229 — 모바일 추천 모달 썸네일 축소

## 배경
- 모바일 `/me` 상품 보기 모달에서 첫 카드 이미지가 전체 폭의 정사각형으로 렌더링되어, 차익/시세/상태 정보가 첫 화면 아래로 밀렸다.
- 이 화면의 핵심은 사진 감상이 아니라 "이 매물을 살지 말지" 판단하는 차익/시세/리스크 정보다.

## 결정
- 추천 모달의 매물 이미지 영역을 모바일에서도 104px 고정 썸네일로 바꿨다.
- 모바일 카드도 `104px + 정보 영역` 2열 레이아웃을 사용한다.
- `sm` 이상은 기존 132px, `lg` 이상은 150px을 유지해 데스크톱 밀도는 크게 바꾸지 않았다.

## 보류
- 초소형 화면에서 차익 카드 텍스트가 답답하면 다음 단계에서 차익 폰트 크기와 배지 줄바꿈을 별도로 조정한다.
