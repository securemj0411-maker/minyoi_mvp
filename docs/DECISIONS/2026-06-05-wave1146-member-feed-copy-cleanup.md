# 2026-06-05 Wave 1146 — 멤버 피드/네비 선공개 잔여 카피 제거

## 결정
- 추천 피드는 멤버십 승인 사용자만 접근하는 화면이므로, 피드 안에서 다시 `멤버십 승인 후`나 `선공개`를 강조하는 안내를 제거한다.
- 네비게이션과 계정 패널에서 `선공개` 표현을 빼고 멤버십/정원 표현으로 정리한다.

## 구현
- `src/components/explore-client.tsx`
  - 피드 끝 안내 카드의 `승인된 멤버에게만...` 문구를 `새로 잡힌 후보가 있으면 아래에 이어서 붙어요.`로 변경했다.
  - `상세 분석은 선공개 멤버십 승인 후...` 안내 줄을 제거했다.
  - 남아 있던 paywall fallback 문구의 `선공개 멤버십` 표현을 `멤버십`으로 변경했다.
- `src/components/app-nav.tsx`
  - 데스크탑 로고 옆 `선공개` 배지를 제거했다.
  - 모바일/게스트 네비 caption의 `선공개 300명`을 `300명 정원`으로 변경했다.
- `src/components/account-panel.tsx`
  - 승인 상태 표시를 `선공개 멤버십`에서 `멤버십 활성화`로 변경했다.
