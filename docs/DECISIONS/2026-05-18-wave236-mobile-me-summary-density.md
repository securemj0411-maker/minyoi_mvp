# 2026-05-18 Wave 236 — /me 모바일 요약 밀도 조정

## 배경

사용자 요청: 모바일 `/me`에서 내 추천 보관함 상단의 `표시 중 / 판매중 / 평균 차익 / 시세 마감` 카드가 세로 공간을 많이 차지해, 정작 수익나는 추천 카드가 한눈에 들어오지 않는다.

## 결정

- 모바일에서는 상단 숫자 요약을 핵심 콘텐츠보다 낮은 우선순위로 본다.
- 데스크톱은 기존 4개 KPI 카드를 유지한다.
- 모바일은 `판매중`, `평균 차익`, 필요 시 `마감`만 한 줄로 보여주고, 세부 숫자는 접이식 요약 안에 둔다.
- 검색/정렬/카드 목록은 기존 흐름을 유지한다. 이번 변경은 카드 노출 전 세로 낭비를 줄이는 데 한정한다.

## 반영

- `UserRevealDashboard`에 모바일 전용 compact summary `<details>`를 추가했다.
- 기존 4개 KPI grid는 `sm` 이상에서만 노출되게 바꿨다.
- `/me` 계약 테스트에 모바일 compact summary가 유지되는지 검증을 추가했다.

## 보류

- 검색/정렬 컨트롤까지 sticky 또는 bottom sheet로 옮기는 작업은 별도 UX 실험으로 둔다.
- `/me` 상단 전체 IA 재배치는 이번 wave에서 하지 않는다.
