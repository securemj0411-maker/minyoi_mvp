## 변경

### 1. 모달 수직 가운데 정렬
- 모달 컨테이너 `items-start sm:items-center` → **`items-center`** (모바일도 가운데).
- `pt-[safe-area+16px]` → `py-[safe-area+16px]` (상하 대칭).
- 안전성: 모달 내부가 `max-h-[calc(100dvh-32px)] + overflow-y-auto`(자체 스크롤)라 긴 단계도
  위 잘림 없이 가운데 OK.

### 2. 송금방법 다시선택 위치 이동
- 결제수단 영역(금액/수단 grid) 안 → **입금했어요 + 신뢰배지 아래**로 이동.
- 위계: 입금했어요(primary CTA) → 신뢰배지 → 송금방법 다시선택(작은 회색 보조).

→ 입금 단계 최종 순서:
```
[입금 금액 / 계좌·토스]
[입금했어요]  (크게)
✓ 사업자 ✓ 실명 ✓ 환불  (배지)
송금 방법 다시 선택  (작게, 보조)
[사업자 정보 확인 ⌄]
[기간/금액 변경] [예약 취소]
```
