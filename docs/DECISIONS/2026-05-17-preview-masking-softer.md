## 박은 변경 (commit `c9484f8`)

### maskName regex 완화
```ts
// 이전:
return `${trimmed.slice(0, 4)}***** *****`;

// 새:
return name.trim().replace(/\d+/g, (m) => "*".repeat(m.length));
```

## 식별 방어

- 숫자 = 모델 세대 / 용량 / 사이즈 / 색상 코드 → 가림
- 단어 = 브랜드 / 모델 라인업 / 특성 → 그대로
- 식별 시도: "갤럭시 S 시리즈 울트라" 까지만 알 수 있음 — 세대 (S22~S25) 불확실, 용량 (256/512) 불확실
- 동기 강한 사용자 = 가격대 + 카테고리 조합으로 좁힐 수 있지만, 사용자 명시 "마스킹 가능성 감안"

## Trade-off

- 식별 가능성 약간 ↑ (단어 정보 노출)
- 사용자 신뢰 큰 ↑ (실제 매물 list 느낌)
- balance: 신뢰 win > 식별 risk
