# 2026-05-18 wave257 — 모바일 네비 크레딧 독립 액션

## 결정
- 모바일 로그인 상태의 상단 네비에서 `대시보드` 버튼과 크레딧 표시를 분리했다.
- 크레딧 표시를 별도 `/plans` 링크로 만들어, 크레딧만 눌러도 충전/요금제 페이지로 이동하게 했다.
- 크레딧 링크에는 현재 보유 수량을 유지하고 접근성 라벨에 `충전하기` 목적을 명시했다.

## 보류
- 모바일 상단에서 드롭다운 충전 패널을 여는 방식은 좁은 폭에서 충돌 가능성이 있어 보류했다.
- 실제 단품 크레딧 구매 상품은 아직 없으므로 `/plans` 구독/요금제 페이지로 연결한다.
