# 2026-05-20 — wave395 `/me` 모바일 상세 PDF 스타일 반영

## 결정

- 사용자 요청에 따라 `/me`의 실제 상품 상세 모달(`PackRevealModal`) 모바일 첫 화면을 첨부 PDF/스크린샷 톤에 맞춰 조정했다.
- 모바일에서는 상품 사진을 기존 290px 카드형 영역에서 풀블리드 히어로 이미지로 키우고, 상단은 검정 이미지 배경/그라데이션/사진 페이지 점/상태 pill/크게 보기 pill을 우선 노출한다.
- 사진 아래 정보 영역은 둥근 상단 시트 형태로 겹쳐 올리고, `AI 판단 · 매물 설명(텍스트) 기준 · 사진은 직접 확인 권장` 카피, 상품명, `득템 점수`, 예상 순익 카드가 바로 이어지게 했다.
- 추가 피드백에 따라 `AI 판단 · 매물 설명(텍스트) 기준 · 사진은 직접 확인 권장` 카피를 제목 왼쪽 묶음 안이 아니라 상품명/득템 점수 행 전체 위 안내 라벨로 이동했다.
- 추가 피드백에 따라 모바일 `득템 점수`를 상품명 오른쪽 행에서 더 위로 올려, AI 판단 안내문 오른쪽 상단에 붙도록 조정했다. 데스크톱은 기존 제목 행 우측 점수 배치를 유지한다.
- 추가 피드백에 따라 `예상 순익` 카드는 수익 숫자와 계산식/비교 매물 보기 버튼까지만 감싸는 독립 카드로 분리했다. `시세 비교 매물` 리스트는 첨부 이미지처럼 별도 섹션 헤더와 흰 리스트 카드로 빼고, 리스크, 비용 확인, 신뢰 Q&A, 판매 채널 비교, 판매 도우미도 같은 초록 카드 안에 섞지 않고 아래 별도 섹션으로 흐르게 한다.
- 추가 피드백에 따라 `최종 매입가 체크`도 첨부 이미지처럼 독립 흰 카드로 재구성했다. 상단 최종 매입가/배송비 반영 pill, `구매 비용`/`리셀 비용` 구분, 점선 구분선, 하단 연녹색 `= 예상 차익` 결과 박스를 한 카드 안에 배치한다.
- 추가 피드백에 따라 `구매 전 점검`, `구매 전 확인`, `어디에 팔지` 섹션의 추가 좌우 inset을 제거해 위아래 카드와 같은 가로 폭으로 맞췄다. 하단 `번개장터 원본 매물 보기` CTA는 왼쪽 N 아이콘을 절대 배치로 바꾸어 버튼 텍스트가 전체 버튼 기준 중앙에 오도록 조정했다.
- 데스크톱/태블릿 모달의 2열 분석 구조는 유지하고, 모바일 전용 class 중심으로 변경해 기존 큰 화면 UI 영향은 최소화했다.
- `BunjangSourceBadge`, `DanawaSourceBadge`, `marketSourceBadge`, 닫힌 상세 설명 state처럼 현재 미사용인 잔여 코드를 제거해 해당 파일 lint warning을 정리했다.

## 보류
