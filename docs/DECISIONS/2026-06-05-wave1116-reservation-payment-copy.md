# 2026-06-05 Wave 1116 - Reservation Payment Copy

## Decision

- 기간 선택 화면에서 "선택 즉시 1석을 7분 동안 예약" 설명 문구를 제거했다. 사용자는 이미 자리 차지 CTA를 눌렀기 때문에 같은 설명을 반복하지 않는다.
- 예약 완료 화면에서 7분 안내 카드와 진행 바를 제거했다. 핵심 정보는 선택 기간, 입금 금액, 계좌, 입금했어요 버튼 순서로 먼저 보이게 한다.
- 예약 지역 표시는 로컬 샘플 동 단위가 아니라 사용자가 선택한 티오 지역 기준으로 고정했다. 예: `동작구 상도제1동 자리`가 아니라 `동작구 자리`.
- `입금했어요` 버튼 자체에 5분 자동 승인 의미를 붙이고, 버튼 아래에 짧은 보조 문구만 남겼다.

## Deferred

- 실제 모바일 인증 세션에서 단계별 스크린샷 검증은 보류했다. 비로그인 로컬 브라우저는 신청 플로우 3단계까지 진입하지 못한다.
