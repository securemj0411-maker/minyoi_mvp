# 2026-05-18 wave255 — 네비 크레딧 충전 진입점 강화

## 결정
- 데스크톱 네비의 크레딧 배지를 클릭 가능한 버튼으로 바꾸고, 보유 크레딧/충전/요금제 관리 패널을 열도록 했다.
- 패널의 `크레딧 충전하기`, `요금제 비교 / 구독 관리` 액션은 `/plans`로 연결했다.
- 모바일 로그인 상태의 상단 `대시보드` 버튼 옆에 현재 크레딧 수를 함께 표시했다.
- 모바일 드로어 하단의 크레딧 박스도 `/plans`로 가는 충전 진입점으로 변경했다.

## 보류
- 크레딧 단품 구매와 구독형 충전 분리는 아직 별도 상품/결제 정책이 필요하므로 보류했다.
- 크레딧 차감/지급 로직과 요금제 schema는 변경하지 않았다.
