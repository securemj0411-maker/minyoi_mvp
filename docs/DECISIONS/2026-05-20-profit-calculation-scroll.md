# 2026-05-20 Profit Calculation Scroll

## Context

예상 순익 카드 안 버튼을 `계산 근거 보기`로 정리했지만, 실제 클릭 동작은 기존 상세 모드 토글에 가까웠다.

사용자는 하단의 `최종 매입가 체크`와 비용 분해 영역이 실제 수익 계산 근거라고 지적했다.

## Decision

- 예상 순익 카드 버튼 문구를 `수익 계산 근거 보기`로 명확히 한다.
- 클릭 시 하단 `최종 매입가 체크` / 비용 분해 영역으로 부드럽게 스크롤한다.
- 클릭 시 상세 모드도 켜서 주변 근거 정보가 더 펼쳐진 상태가 되도록 유지한다.

## Deferred

- 비용 분해 영역을 기본 접힘으로 바꾸는 방식은 보류했다. 현재는 첫 화면 신뢰 근거와 하단 계산 근거가 모두 노출되는 구조를 유지한다.
