# 2026-06-05 Wave 1117 - Deposit Approval Flow

## Decision

- 입금 전 예약 화면은 7분 예약 타이머와 "7분 안에 입금하지 않으면 예약 취소" 안내를 유지한다.
- 사용자가 `입금했어요`를 누른 뒤에는 7분 예약 타이머를 숨기고 5분 자동 승인 타이머만 보여준다. 기간 변경/예약 취소 버튼도 입금 확인 요청 후에는 숨긴다.
- 텔레그램 입금 확인 알림 문구를 "관리자 페이지에서 확인"이 아니라 "로그인 없이 즉시 승인/거절"로 정리했다.
- 텔레그램 승인/거절 링크는 토큰 검증 후 확인 폼 없이 바로 처리한다.

## Deferred

- 텔레그램 앱 내부 브라우저에서 링크 미리보기/오클릭 방지 UX는 별도 장치 없이 즉시 처리로 간다. 운영 편의 우선 결정이다.
