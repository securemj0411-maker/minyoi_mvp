# 2026-05-21 — 쉬운모드 판매 속도 카피 재정의

## 결정
- 쉬운모드 판매 속도는 `빨리 매입하라`는 CTA처럼 보이지 않게 한다.
- 판매 주기는 사용자가 리셀 후 돈이 얼마나 오래 묶일지 가늠하는 정보로 설명한다.
- `최근 등록`은 수요 지표처럼 오해될 수 있어 쉬운모드 판매 속도 카드에서 제거한다.

## 구현
- 판매 속도 제목을 `되팔면 보통 N 안에 팔린 기록이 있어요`로 바꿨다.
- 보조 문구를 `동일 모델 하루 평균 판매량`과 `돈이 얼마나 오래 묶일지` 중심으로 수정했다.
- 쉬운모드 판매 속도 시각 카드에서 3열을 2열로 줄이고 `최근 등록` 지표를 제거했다.
- 계약 테스트에 최근 등록 문구가 쉬운모드 상세 소스에 다시 들어오지 않도록 조건을 추가했다.

## 보류
- 판매 속도 계산식과 데이터 소스는 바꾸지 않았다. 이번 변경은 사용자-facing 문구와 쉬운모드 표시 방식만 다룬다.
