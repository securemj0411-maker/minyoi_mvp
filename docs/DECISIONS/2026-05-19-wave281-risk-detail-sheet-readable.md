# 2026-05-19 Wave 281 — 위험 신호 설명을 읽기 쉬운 시트로 확대

## 결정
- `/me` 상품 상세에서 안전/주의 설명을 눌렀을 때 뜨는 작은 popover를 모바일 바텀시트형 설명 패널로 키웠다.
- 글자 크기를 9~11px 중심에서 12~14px 중심으로 올리고, 5축 위험 신호를 줄글 대신 카드형 체크 항목으로 보여준다.
- 사용자에게 노출되던 `hard-block`, `POOL_BLOCK_FLAGS` 같은 운영자 용어는 제거하고 "강한 차단 신호는 먼저 걸렀고, 남은 확인 포인트만 보여준다"는 말로 바꿨다.

## 보류
- 안전 설명을 별도 상세 페이지나 FAQ와 직접 연결하는 작업은 보류했다.
- 각 위험 축별 실제 예시/사진 가이드를 붙이는 확장도 추후 UX 흐름이 잡힌 뒤 진행한다.
