# 2026-05-20 Wave 426 — 쉬운모드 판매 속도 문구 정리

## 결정
- 쉬운모드 마지막 판매 속도 단계에서 `판매완료 누적`, `판매완료 표본`, `거래완료 표본` 같은 내부 분석 용어를 쓰지 않는다.
- 사용자가 바로 이해하도록 `하루 평균 팔림`, `최근 7일 기준 하루 약 N개 팔림`, `비슷한 거래 기록`으로 표현한다.

## 구현
- `dailySoldCountLabel()`을 추가해 최근 7일 거래 수를 하루 평균 개수로 변환한다.
- velocity가 있으면 쉬운모드 카드의 두 번째 지표를 `하루 평균 팔림`으로 보여준다.
- velocity가 부족한 경우에도 `표본` 대신 `비슷한 거래 기록` 또는 `판매 기록 확인 중`으로 보여준다.

## 보류
- 판매 속도 계산식 자체는 바꾸지 않았다. 이번 변경은 사용자-facing 문구와 쉬운모드 표시 방식만 다룬다.
