# 2026-06-04 Wave 1092 - Feed Member Offer Contract

## 결정

- 피드의 `Member-only offer` 배너가 현재 가입 기간을 모른 채 1개월용 문구를 고정 노출하던 구조를 제거했다.
- `/api/membership/status`가 최신 승인 멤버십 신청의 활성 플랜 정보를 내려주도록 확장했다.
- 피드 배너는 활성 플랜 기준으로만 노출한다.
  - 1개월권: 3개월/6개월/12개월 첫 연장 특가
  - 3개월권: 6개월/12개월 전용 특가
  - 6개월권: 12개월 전용 특가
  - 12개월권: 배너 미노출
- 피드 오퍼 CTA는 `/plans`로 보내지 않고, 피드 카드 안에서 바로 연장 예약과 입금 확인 요청까지 처리한다.

## 보류

- 별도 전용 URL 이벤트 페이지는 만들지 않았다. 현재 wave에서는 사용자가 피드에서 버튼을 눌러야만 열리는 카드 내 이벤트 플로우로 대체했다.
- 이벤트 문구/가격 정책은 `membership-plans.ts`의 현재 업셀 상품을 기준으로 사용한다.
