# 2026-06-04 Wave 1098 — 피드 하단 피드백 활동 제거

## 결정

- 멤버십 전환 이후 `/me` 추천 피드 하단에 남아 있던 `MyFeedbackActivity` 위젯 렌더를 제거했다.
- 해당 위젯은 과거 토큰 보상제 문구(`토큰 +3`)를 노출하고 있어 현재 멤버십 상품 톤과 맞지 않았다.

## 보류

- 신고/피드백 API와 운영자 검수 화면은 즉시 삭제하지 않았다. 알고리즘 보정 데이터 수집이나 운영자 내부 검수 용도로 재정의할 수 있어, 사용자 피드 하단 노출만 먼저 제거했다.
- 상세 페이지/모달 안의 정보 오류 신고 문구는 별도 UX로 재설계가 필요하다. 이번 작업 범위는 사용자가 지적한 피드 하단 레거시 노출 제거로 제한했다.
