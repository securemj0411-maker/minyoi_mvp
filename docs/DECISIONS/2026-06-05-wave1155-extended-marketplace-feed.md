# Wave 1155 - 중고나라/번개장터 확장 버튼 서버 모드 분리

## 결정
- 피드 하단의 `중고나라·번개장터까지 보기` 버튼을 단순 `source=all` 전환이 아니라 `marketplaces=extended` API 모드로 분리했다.
- 확장 모드에서는 당근 근처 후보 prefetch를 끄고, 중고나라/번개장터 후보를 우선 quota로 확보한 뒤 나머지 슬롯을 채운다.
- 응답 정렬도 확장 모드에서는 중고나라/번개장터를 먼저 보여주고, 일반 피드에서는 기존 당근 거리 우선 정렬을 유지한다.

## 이유
- 기존 구조는 프론트가 `source=all`로 바꿔도 서버가 source 미지정 요청으로 해석해 당근 근처 후보를 먼저 밀었다.
- 사용자는 버튼을 눌렀을 때 중고나라/번개장터가 즉시 섞여 나오길 기대하므로, 버튼 의도를 서버 후보 구성에 명시해야 한다.
