# 2026-05-22 Wave 533 — 쉬운모드 상세 리포트 전환 선택지

## 결정
- 쉬운모드에서 `상세 숫자 리포트 보기` 또는 마지막 `이 매물 자세히 보기`를 누를 때 바로 상세 화면으로 닫지 않고, 전환 방식을 한 번 더 묻는 선택 sheet를 추가했다.
- 선택지는 `이번만 상세 리포트 보기`와 `앞으로 상세 리포트를 기본으로 보기`로 나눴다.
- 기본 상세 모드로 바꿔도 상단 `쉽게 보기` 버튼으로 언제든 다시 쉬운모드를 열 수 있다는 안내를 같이 노출한다.

## 구현
- `pack-reveal-modal.tsx`에 `DetailReportModeChoiceSheet`를 추가했다.
- 쉬운모드 skip/complete 이벤트는 선택 확정 시점에 기록하도록 옮겼고, `defaultDetailed` 메타를 붙였다.
- 현재 열려 있는 상세 카드도 선택 직후 상세 모드로 전환되도록 `minyoi:modal-mode-changed` client event를 연결했다.

## 보류
- 상세 카드 내부의 `수익 계산 근거 보기` 버튼은 기존처럼 상세 모드를 기억하는 동작을 유지했다. 이번 변경 범위는 쉬운모드 exit CTA에 한정한다.
