# 2026-06-05 Wave 1123 - Membership Approval and Location Flow

## Decision

- 텔레그램 멤버십 승인 링크는 세션 없이 바로 처리되어야 하므로, HMAC 검증 외에 신청 row에 기록된 텔레그램 액션 토큰도 fallback으로 확인한다.
- 로컬/운영 환경이 섞여 알림 링크가 생성되어도 이후 링크 검증이 깨지지 않도록, `입금했어요` 알림 생성 시 승인/거절 토큰을 신청 row의 `admin_note`에 함께 남긴다.
- 멤버십 기간 선택 후 자리 확보 요청 중에는 전체 화면 진행 오버레이를 띄워 사용자가 멈춘 화면으로 오해하지 않게 한다.
- GPS 위치 확인은 곧바로 세부 지도에 진입하지 않고, 먼저 `동네 맞나요?` 확인을 받은 뒤 세부 지역 티오 선택으로 이어지게 한다.

## Deferred

- 이미 발송된 과거 텔레그램 링크 중 DB에 토큰이 기록되지 않은 링크는 복구하지 않는다. 새로 생성되는 링크부터 fallback 검증이 적용된다.
