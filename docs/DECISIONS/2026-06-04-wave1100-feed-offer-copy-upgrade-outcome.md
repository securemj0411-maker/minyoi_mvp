# 2026-06-04 Wave 1100 — 멤버 전용 업셀 문구를 결과 중심으로 변경

## 결정

- 피드 상단 멤버 전용 1시간 업셀에서 `차액`, `부족한 기간`, `1년권에 가깝게` 같은 내부 계산식 문구를 제거했다.
- 사용자가 바로 이해할 수 있게 `단 N원으로 1년 무제한 멤버십 업그레이드`처럼 결과 중심 문구로 바꿨다.
- 6개월 대상 업셀도 같은 방식으로 `6개월 무제한 멤버십 업그레이드`를 표시한다.

## 보류

- 실제 업그레이드 계산 로직은 변경하지 않았다. 이번 작업은 결제 제안 카피와 플랜 메타데이터 표현 정리다.
