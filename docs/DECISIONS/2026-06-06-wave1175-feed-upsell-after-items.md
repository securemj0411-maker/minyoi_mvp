# 2026-06-06 Wave 1175 — 피드 업셀 노출 순서

## 결정
- 피드 멤버 전용 업셀 카드는 로딩 중/빈 상태/스크랩 모드에서는 숨긴다.
- 실제 추천 매물이 1개 이상 렌더된 뒤, 상품 리스트 아래에 이어서 노출한다.

## 이유
- 기존에는 피드 상품이 느리게 로딩되는 동안 업셀 카드가 먼저 보여서, 사용자가 “매물 보러 왔는데 결제 제안부터 뜬다”고 느낄 수 있었다.
- 피드의 1차 목적은 추천 매물 확인이므로, 업셀은 매물을 본 뒤의 추가 제안으로 배치하는 편이 덜 혼란스럽다.

## 보류
- 업셀을 몇 번째 상품 뒤에 삽입할지는 이번에 세분화하지 않았다. 실제 피드 체류/전환 데이터를 보고 3~6번째 카드 뒤 삽입을 별도 실험한다.
