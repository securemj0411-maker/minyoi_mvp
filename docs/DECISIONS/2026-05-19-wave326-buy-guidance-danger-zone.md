# 2026-05-19 Wave 326 — 협상 가이드 위험 구간 명시 + 동사 명시

사용자 지적:
1. "**손해 시작 740,270원 (이 이상은 손해)**" — "이 이상에 사면" 동사 빠짐. "74만원 이상이 손해?" 헷갈림.
2. 협상 시도 목표가 단방향 (더 깎으면 좋음)만 있고 **위험 구간**(차익 1만 미만) 표시 없음. 사용자가 손익분기까지 사도 OK라고 오해 가능.

## 결정

### 1. 가격대별 의미 4단 구조

```
현재 가격 700,000원 — 남는 돈 +40,270원

↓ 협상 시도 688,000원      +52,000원 (더 좋음)
⚠ 730,270원 이상에 사면    +1만원 미만 (위험)
✕ 740,270원 이상에 사면    손해 (수익 0 이하)
```

- **현재 가격** — 헤드라인 박스 (emerald bg)
- **협상 시도** — 더 좋은 가격 (↓ 표시)
- **위험 구간** — 차익 1만원 미만 시작 가격. amber (⚠)
- **손해 구간** — 손익분기. rose (✕)

### 2. 동사 명시
"이 이상은 손해" → **"이 이상에 사면 손해"** (사용자 지적 그대로)

### 3. 위험 임계 신규 추가
`buy-price-guidance.ts`:
- `DANGER_PROFIT_THRESHOLD = 10000` 상수
- `dangerStart = breakEven - 10000` — 차익 1만원 미만 시작 가격
- `negotiationProfit = breakEven - negotiationTarget` — 협상 도달 시 남는 돈

차익 1만원 미만 = 사실상 손해 (잡비 + 시간 비용 + 시세 변동 흡수 불가).
