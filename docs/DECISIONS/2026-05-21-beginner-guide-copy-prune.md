# 2026-05-21 — 쉬운모드 보조 설명 정리

## 결정
- 비교 매물 화면의 상태 기준 반복 카드를 제거했다.
- 시세 흐름, 매입가, 되팔 비용, 판매 채널 단계의 value note 문구를 제거했다.
- 판매 속도 표본 부족 카드는 파란 포인트 숫자 카드로 바꿨다.
- 구매 전 체크 단계는 설명성 문구를 없애고, 확인 질문 개수 안내를 본문 위치로 올렸다.
- 시세 그래프의 데이터 누적 배너를 제거했다.

## 배경
- 쉬운모드의 목적은 한 화면에 한 판단만 보여주는 것인데, 보조 설명이 중복되어 모바일에서 피로감이 커졌다.

## 보류
- 각 단계의 본문 길이는 실제 모바일 캡처 기준으로 계속 줄인다.
