# 2026-05-19 Wave 316 — 상세 헤드라인 순익 기준 분리

## 결정
- `/me` 상품 상세 헤드라인의 `현재 차익` 문구를 `예상 순익`으로 바꿨다.
- 헤드라인 퍼센트는 `순익률`로 명시하고, 시세-매입가 차이인 `총차익`은 작은 보조 chip으로 분리했다.
- 회수 속도 표본이 있는 경우 `+원/일` 형태의 일 단위 회수 효율을 추가했다.
- 회수 표본이 부족하면 일수익을 추정하지 않고 `일수익 표본 부족`으로 표시한다.

## 보류
- KREAM/솔드아웃/중고나라/당근 시세 라인은 데이터 수집 및 약관/비용 확인 전까지 붙이지 않는다.
- 가품 의심도, 역이미지 검색, 판매자 계정 나이/분쟁률은 현재 데이터 모델에 없어서 별도 수집 설계가 필요하다.
- 신발 사이즈별 수요, 체결률, 사용자별 ROI 학습 루프는 장기 해자 과제로 분리한다.
