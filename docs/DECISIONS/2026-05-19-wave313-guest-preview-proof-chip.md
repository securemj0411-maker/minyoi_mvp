# 2026-05-19 Wave 313 — 비로그인 프리뷰 카테고리 힌트 제거

## 결정
- 비로그인 메인 프리뷰 카드에서 신발/가전/스마트워치 같은 카테고리 텍스트, 카테고리 아이콘, 카테고리별 색상 힌트를 제거한다.
- 카드마다 근거는 여러 개 나열하지 않고 `후기 N+ 셀러`, `평점 5.0 셀러`, `평균 N일 회전`, `시장 표본 N건` 중 하나만 우선순위로 보여준다.
- 프리뷰 API는 셀러 평점/후기 수를 함께 내려줘 신뢰 근거 chip을 구성한다.
