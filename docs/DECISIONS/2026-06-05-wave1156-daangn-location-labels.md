# Wave 1156 - 당근 피드/상세 위치 표기 보강

## 결정
- 당근 피드와 상세 헤더의 위치 표기를 동 단독에서 `구/시 + 동` 중심의 축약 주소로 바꿨다.
- `directTradeLocation`이 있으면 피드에서도 거리 라벨과 합쳐 `동작구 흑석동 · 약 1km`처럼 보여준다.
- 상세 헤더는 기존 local helper 대신 공용 `compactTradeLocationLabel`을 사용한다.

## 이유
- 현재 당근 후보 수집은 기본 10km 반경이고, 실행 가능 판정도 `near <= 6km`, `reachable <= 10km`라서 동 이름만 보이면 같은 구/다른 구를 오해하기 쉽다.
- 특히 서울/수도권은 1~2km 안에서도 구가 바뀔 수 있으므로 피드에서 최소 `구 + 동` 정보가 필요하다.

## 보류
- 거리 반경 자체를 1~2km로 줄일지는 별도 정책 결정으로 남긴다. 지금은 노출 정보 개선만 적용했다.
