# 2026-06-06 Wave 1167 — 결제 모달 2단계 분리

## 결정
- 멤버십 신규/연장 입금 모달을 `입금 방법 선택`과 `입금 확인` 두 단계로 분리했다.
- 첫 화면에서는 토스 송금과 계좌송금 두 선택지만 크게 보여주고, 선택 후에만 입금 금액·계좌 정보·사업자 확인·입금했어요 CTA를 노출한다.
- 피드 내 멤버 전용 업셀 예약도 같은 패턴으로 맞춰 결제 흐름이 화면마다 다르게 보이지 않게 했다.

## 보류
- 토스 실제 결제 성공 콜백은 아직 없어서 사용자가 송금 후 `입금했어요`를 눌러야 하는 구조는 유지한다.
- 입금 확인 자동화는 기존 5분 자동 승인/폴링 흐름을 유지하고, 별도 오픈뱅킹 연동은 추후 검토한다.
