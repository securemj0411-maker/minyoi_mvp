# 2026-06-05 Wave 1134 - Feed Velocity and Comparison Copy

## 결정
- 피드 회전 라벨을 상세와 맞춰 `평균 N일 회전`으로 통일했다.
- 회전 의미는 피드 카드의 `?` 도움말로 설명하게 했다.
- 비교 매물 리스트의 `+N%`, `-N%`, `비슷` 표시는 수익률로 오해될 수 있어 제거했다.
- 더 찾아보기 로딩 문구는 `받는 중`에서 `가져오는 중`으로 바꿨다.

## 보류
- 회전 신뢰도 high/medium/low를 별도 뱃지로 노출하는 작업은 보류했다.
- 비교 매물별 가격 차액 표시는 재도입하지 않는다. 필요하면 별도 설명 UI를 먼저 설계한다.
