# Wave 509 — 상세페이지 그래프 위치와 비교 매물 정렬

## 문제
- 상세페이지에서 쉬운모드에 없는 시세 그래프가 비교 매물보다 아래에 있어 가치 정보의 위계가 약했다.
- 비교 매물 정렬 코드는 낮은 가격순 주석/구현으로 남아 있어, 사용자가 기대하는 고가 우선 비교와 어긋날 수 있었다.

## 결정
- 상세 숫자 페이지에서는 시세 그래프/시장 분석을 비교 매물 바로 위에 둔다.
- 비교 매물은 `market-source` API에서 outlier가 제거된 결과를 받은 뒤 가격 높은 순으로 표시한다.
- 쉬운모드의 입문 설명과 상세모드의 데이터 탐색 역할을 분리한다.

## 구현
- `MarketHistoryChart`, `MarketGraphTrustLine`, `SkuListingFlowMini` 묶음을 `ComparableListingsPanel` 바로 위로 이동했다.
- `ComparableListingsPanel` 정렬을 가격 내림차순으로 고정했다.

## 보류
- 그래프와 비교 매물을 하나의 탭/접힘 UI로 합치는 개편은 보류했다. 지금은 순서만 바로잡아 상세페이지 탐색 흐름을 개선한다.
