# 2026-05-18 Wave 263 — /me 시세 그래프 출처 라벨 명확화

## 문제
- /me 상품 모달의 시세 그래프 범례가 `호가`, `거래가`만 표시되어 사용자가 번개장터 기준인지, 다나와 기준인지 바로 알기 어려웠다.
- 특히 다나와 기준선이 함께 보이는 미개봉 상품에서는 출처가 섞여 보이면 신뢰 설명이 약해진다.

## 결정
- 번개장터 데이터 기반 그래프 제목을 `번개장터 시세 30일 추이`로 바꿨다.
- 범례와 하단 요약을 `번개장터 호가`, `번개장터 거래가`, `번개장터 거래 N건`처럼 출처가 드러나게 바꿨다.
- 다나와 reference 상품은 제목을 `다나와 · 번개 미개봉 추이`로 두고, 번개장터에서 온 선은 `번개 미개봉 호가/거래가`로 분리했다.

## 보류
- 그래프 위에 별도 출처 툴팁을 붙이는 것은 모달 밀도와 겹치므로 보류했다. 필요하면 `그래프 기준 보기` 상세 안에 더 풀어 설명한다.
