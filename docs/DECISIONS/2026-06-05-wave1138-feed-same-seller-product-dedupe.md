# 2026-06-05 Wave 1138 - Feed Same Seller Product Dedupe

## 결정
- 추천 피드 응답에서 `같은 셀러 + 같은 출처 + 같은 comparable_key + 같은 상태` 매물은 대표 1개만 남긴다.
- 대표 매물은 가격이 낮은 것을 우선하고, 가격이 같으면 최근 검증된 매물을 우선한다.
- 중복 매물을 DB에서 `invalidated` 처리하지 않고, 피드 응답 단계에서만 숨긴다.

## 이유
- 같은 셀러가 가격을 낮춰 재등록한 경우는 분석 자료로 의미가 있으므로 원본/ready 기록을 삭제하면 안 된다.
- 하지만 피드에 같은 상품이 2~3개 동시에 뜨면 사용자에게 도배/버그처럼 보인다.

## 보류
- 가격 인하 히스토리를 상세 페이지에 별도 표시하는 기능은 보류했다.
- same seller duplicate suppression 통계를 운영자 화면에 노출하는 작업도 보류했다.
