# 2026-05-18 Wave 223 — 번개장터/다나와 브랜드 로고 표시

## 문제
- `번개장터 열기`, `다나와`, `번개 S급` 같은 출처 표시가 텍스트/이모지 위주라 사용자가 어떤 외부 출처인지 즉시 알아보기 어렵다.

## 결정
- 사용자가 제공한 번개장터/다나와 로고를 `public/brand/`에 추가한다.
- `/me` 카드, 상품 보기 모달, 시세 근거 모달, 운영자 풀의 시세 출처/외부 링크 표시에서 로고를 함께 보여준다.
- 다나와 reference anchor는 초록 `d` 로고, 번개장터 링크/번개 시세는 번개 로고로 통일한다.
