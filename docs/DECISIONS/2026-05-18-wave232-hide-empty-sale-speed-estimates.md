# 2026-05-18 Wave 232 — 판매 속도 표본 부족 문구 정리

## 배경
- 상품 보기 모달에서 판매 기록이 0건인 매물에 `비슷한 상품은 보통 — 안에 팔렸어요` 문구가 표시됐다.
- 원인은 판매 속도 추정 시간이 `null`인데 UI가 문장을 그대로 조립한 것이다.

## 결정
- 판매 속도 추정 시간이 없거나 판매 표본이 0건이면 시간 문장을 만들지 않는다.
- 대신 `판매 속도 표본 부족`과 최근 판매/현재 판매중 표본 수만 표시한다.
- 가격대별 회전 그래프도 추정 시간이 없으면 `— 안에 팔렸어요`를 노출하지 않고 짧은 안내 카드로 대체한다.

## 보류
- 판매 속도 데이터 자체를 보강하는 것은 market/velocity worker 표본 확충 과제로 남긴다.
