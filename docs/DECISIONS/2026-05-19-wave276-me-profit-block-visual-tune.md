# 2026-05-19 Wave 276 — /me 현재 차익 블록 톤다운

## 문제
- /me 목록 카드와 상품 모달의 `현재 차익` 영역이 초록 배경/초록 테두리/큰 굵은 숫자로 동시에 강조되어 촌스럽고 무거워 보였다.
- 사용자가 원하는 방향은 기능 강조는 유지하되, 숫자 자체만 밝은 초록으로 살리고 나머지 표면은 담백하게 낮추는 것이다.

## 결정
- 카드 배경을 emerald 계열 면색에서 흰색/크림 계열 중립 표면으로 바꿨다.
- 강조는 왼쪽 얇은 accent bar와 차익 숫자의 밝은 green(`#00a862`, dark `#5dffae`)만 사용한다.
- `현재 차익` 라벨은 uppercase/자간 강조를 빼고 일반 보조 텍스트처럼 낮췄다.
- 수익률 pill도 amber 강조를 줄이고 중립 톤으로 바꿨다.
- 모달 숫자 크기를 `text-xl/sm:text-2xl`에서 `text-lg/sm:text-xl`로 낮춰 그래프와 상세 정보가 더 빨리 보이게 했다.

## 보류
- 전역 폰트 변경은 보류했다. 현재 사이트는 이미 Pretendard 기반이라, 이번에는 폰트 패밀리보다 weight/색/표면 정리로 Claude/Toss에 가까운 차분한 톤을 우선 적용했다.
