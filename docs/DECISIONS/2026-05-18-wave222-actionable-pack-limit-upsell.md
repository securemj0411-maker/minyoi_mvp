# 2026-05-18 Wave 222 — 추천 한도 안내를 업그레이드 행동으로 연결

## 문제
- 추천 상품 수 조절 영역에서 `한도 도달`, `월 4/5`, `오늘 1/1`, `부족 2`처럼 상태만 보여주고 다음 행동을 명확히 안내하지 않았다.
- 일반 사용자는 "그래서 뭘 하라는지" 알기 어렵고, 서비스가 의도적으로 적게 보여주는 느낌을 받을 수 있었다.

## 결정
- 월/일 한도 임박·도달 또는 크레딧 부족 상태를 설명형 안내 박스로 표시한다.
- 일일 한도 도달 시 "내일 다시 열 수 있음"과 "지금 더 보려면 한도 큰 요금제"를 같이 안내한다.
- 한도/크레딧 때문에 진행 불가한 경우 메인 CTA를 `/plans`로 연결되는 업그레이드 행동으로 바꾼다.

## 보류
- 남은 시간 카운트다운(자정 리셋까지 N시간)은 서버 기준 리셋 정책을 UI에 명시한 뒤 추가한다.
