# 2026-06-04 Wave 1096 - Feed Offer Modal Flow

## 결정

- 피드 멤버 이벤트 카드에서 계좌/입금확인 영역을 inline으로 펼치지 않는다.
- 피드 카드는 짧은 제안과 `제안 수락` CTA만 보여준다.
- `제안 수락`을 누르면 하단/중앙 모달에서 차액, 계좌, `입금했어요` 흐름을 처리한다.
- `예약` 표현은 사용자-facing CTA에서 제거하고 `제안 수락`, `전환 제안` 중심으로 바꿨다.

## 이유

- inline 계좌 영역은 모바일 피드를 과도하게 늘리고, 상품 탐색 흐름을 깨뜨린다.
- `예약`은 사용자가 언제/무엇이 확정되는지 이해하기 어렵다.

## 보류

- 별도 전용 전환 페이지는 만들지 않았다. 현재는 모달이 가장 작은 구조 변경으로 결제 흐름을 분리한다.
