# 2026-05-18 Wave 225 — `/me` 카드는 스캔용으로 단순화

## 문제
- `/me` 카드에 `상품 보기`, `공략 보기`, 위험/근거 chip이 같이 노출되어 상세 모달과 정보가 중복됐다.
- 모바일에서 버튼이 많아지고 카드 스캔성이 떨어졌다.

## 결정
- `/me` 카드의 `상품 보기`와 `공략 보기` 버튼을 제거한다.
- 정상 카드 전체를 클릭/키보드 Enter/Space로 상품 보기 모달이 열리게 한다.
- `/me` 카드에서는 상품명, 등급, 매입/시세/출처, 상태, 현재 차익 정도만 남긴다.
- 위험/신뢰/비교 근거/공략은 상품 보기 모달 안에서 확인하도록 책임을 모은다.

## 보류
- 카드 전체 클릭 전환 후 실제 모바일 사용에서 affordance가 부족하면, 버튼 복귀 대신 카드 가장자리/hover/touch 피드백을 더 강화한다.
