# 2026-05-19 wave307 - 상세 사진 viewport 풀블리드

## 결정
- /me 상세 모바일 상품 사진을 컨텐츠 폭이 아니라 `100vw` 기준으로 렌더링하도록 바꿨다.
- `left-1/2`, `w-screen`, `-translate-x-1/2` 조합으로 스크롤 컨텐츠의 `px-3`와 바깥 여백을 모두 넘겨 실제 화면 좌우 끝까지 채운다.
- `sm` 이상에서는 기존 132px/150px 썸네일 레이아웃으로 되돌린다.

## 보류
- 사진 자체를 `object-cover`로 자르지는 않았다. 실제 상품 일부가 잘릴 수 있어 `object-contain`과 모바일 scale 보정은 유지한다.
