# 2026-06-05 Wave 1125 - Membership Flow Mobile Shell

## Decision

- 멤버십 기간 선택 단계에서는 바깥 온보딩 카드 shell을 제거해 모바일 화면의 중복 라운드 카드 구조를 줄인다.
- 기간 선택 단계에 `뒤로` 버튼을 추가해 지역 FOMO 단계로 돌아갈 수 있게 한다.
- 기간 선택 CTA는 모바일에서 하단 고정 위치로 맞춰 이전 온보딩 단계의 CTA 흐름과 일관성을 유지한다.
- 모바일 햄버거 메뉴는 멤버십 승인 상태를 읽어 승인자에게 `신청하기` 대신 `피드 보기` / `멤버십 관리` 톤으로 노출한다.

## Deferred

- 데스크톱 상단 nav의 `멤버십 신청` 라벨까지 승인 상태에 따라 바꾸는 작업은 별도 UX 정리 때 같이 본다.
