# 2026-05-19 Wave 325 — 매입가 가이드 완전 재설계

사용자 지적: 매물 700,000원 (예상 순익 +40,270원) 상태에서 가이드가 **"5% 마진 — 패스 권장"**으로 표시. 풀에 노출된 매물인데 패스 권장? 모순.

## 진단

### 기존 (잘못 박힌 로직)

- `추천 매입가 = 시세 × 82%` (18% 마진 가정)
- `패스 기준 = 시세 × 90%` (10% 마진 미만이면 패스)
- 미개봉 IT(아이패드)는 평균 마진 5~10% → **거의 모든 풀 매물이 "패스 권장"** 으로 뜸

### 근본 모순

- **우리 풀 정책**: 차익 양수 + 안전 마진 통과한 매물만 노출 (수수료 + 배송 + 버퍼 다 차감 후 남는 매물).
- **사용자 정신 모델**: "이 매물 사도 되나?"는 풀에 있으니 이미 OK. "협상 시 어디까지?"가 진짜 궁금.
- **잘못된 가이드**: 18% 마진 같은 카테고리 무관 기준 → 풀 정책과 정면 충돌.

## 결정 — 가이드 완전 재설계

### 새 프레임 — 협상 가이드

매입가 가이드가 아니라 **"협상 가이드"**. 매물 가격(=매입가)은 이미 알려진 값. 협상 시 어디까지 시도/언제 손 떼나가 핵심.

**3가지 정보만 노출:**

1. **남는 돈 (예상 순익)** — `+40,270원`
   - 풀 통과 매물이면 항상 양수
2. **협상 시도 목표** — `680,000원 (-20,000원 깎으면)`
   - = 현재 가격 - 협상 여유
   - 협상 여유 = `min(currentProfit × 30%, 20,000원)` (보수적)
3. **손해 시작** — `740,270원 (이 이상은 손해)`
   - = 손익분기 (시세 - 비용)

### Verdict 4단계 (차익 절대값 기준)

| 차익 범위 | verdict | 라벨 | 색 |
|---------|---------|------|----|
| 50,000원+ | great | 충분한 차익 · 협상 없어도 OK | emerald |
| 20~50k | good | 괜찮은 차익 · 협상 시 +α | emerald |
| 10~20k | fair | 차익 작음 · 협상 권장 | amber |
| <10k | tight | 차익 매우 작음 · 적극 협상 필요 | amber |

**rose(빨강) verdict 제거** — 풀에 있는 매물은 안전 마진 통과한 매물이라 "패스 권장" 불필요. 차익이 진짜 적으면 amber로 "협상 권장" / "적극 협상" 정도. 풀 정책과 일관.

### 기존 vs 새 — 같은 매물(아이패드 700k, 시세 776k, 차익 +40k)

**기존:**
> 추천 매입가: ~600,603원 (+18% 확보)
> 패스 기준: 662,677원 이상 — 손 떼기
> 현재 5% 마진 — 패스 권장 · 손실 위험
> 손익분기 740,270원

**새:**
> 남는 돈: +40,270원
> 협상 시도 목표: 680,000원 (-20,000원 깎으면)
> 손해 시작: 740,270원 (이 이상은 손해)
> ✅ 괜찮은 차익 · 협상 시 +α

훨씬 명확. 사용자 정신 모델 일치.
