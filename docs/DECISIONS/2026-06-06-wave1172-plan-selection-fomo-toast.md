# Wave 1172 — 신규 기간 선택 화면 FOMO 토스트

## 배경
- 신규 가입자는 지역 티오 화면에서 희소성을 확인한 뒤 멤버십 기간을 선택한다.
- 가격이 노출되는 순간에도 "다른 사람이 자리를 확보하고 있다"는 사회적 증명/압박이 이어져야 전환 동기가 유지된다.

## 결정
- 신규 가입 기간 선택 화면에서만 짧은 social proof 토스트를 노출한다.
- 연장 흐름은 이미 회원의 기간 추가라 지역 티오 압박과 섞지 않는다.
- 화면 수직 공간을 더 먹지 않도록 기간 카드 사이에 고정 요소를 넣지 않고, 상단 floating toast로 2회만 노출한다.

## 보류/주의
- 현재 토스트는 fallback mock 이벤트 기반이다.
- 실제 가입 이벤트와 섞는 고도화는 plans page social proof 통합 시 별도 처리한다.
