# Wave 508 — 중고나라 수수료와 최종 순익 흐름 정리

## 문제
- 채널별 재판매 카드에서 중고나라와 번개장터 순익이 같은 값으로 표시됐다.
- 중고나라 안심결제는 판매자/구매자 부담 구조가 번개장터와 다르므로 같은 3.5% 셀러 차감으로 보여주면 부정확하다.
- 쉬운모드의 비용 카드가 최종 순익을 맨 위에 보여줘 계산 흐름상 어색했다.

## 확인
- 공개 보도 기준 중고나라는 안심결제 판매자 수수료 1%를 부과하고, 2만원 이하 거래는 면제된다.
- 기존 구매자 대상 안심결제 수수료는 유지되는 것으로 보인다.

## 결정
- 채널별 재판매 순익에서 번개장터는 기존 보수 기준 3.5% 셀러 수수료 차감을 유지한다.
- 중고나라는 판매자 1% 차감으로 별도 계산하고, 구매자 수수료는 판매자 순익에서 차감하지 않고 “구매자 3.5% 별도”로 표시한다.
- 최종 순익 카드 순서는 `매입가 → 기준 시세 → 되팔 때 비용 → 최종 예상 순익`으로 바꾼다.

## 보류
- 전체 후보 pool의 기본 expected_profit 계산식은 아직 번개장터 보수 기준 3.5%를 유지한다. 채널별 비교 UI에서 중고나라/당근 보정을 보여주는 방식으로 먼저 적용한다.
