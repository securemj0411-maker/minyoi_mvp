# 2026-05-20 Wave 406 — 상세 모달 사진·예상 순익 밀도 조정

## 배경
- 사용자가 상세 화면에서 예상 순익 숫자가 너무 크고, 상품 사진도 약 30% 줄여도 될 것 같다고 피드백했다.
- 기존 `/me` 상세 모달은 모바일 사진이 거의 정사각 풀폭이고, 데스크톱 사진도 240/280px라 시세 그래프와 근거 정보로 내려가는 흐름을 막았다.
- 예상 순익 카드 숫자도 28px/900으로 시각 우선순위가 과하게 높았다.

## 결정
- `RevealProductImage` 모바일 비율을 `4/4.2`에서 `4/3`으로 바꿔 높이를 약 30% 줄였다.
- 데스크톱 사진은 240/280px에서 168/196px로 낮추고, grid column도 이미지 크기에 맞췄다.
- 예상 순익 숫자는 28px에서 22px로 줄이고 카드 padding/watermark도 낮췄다.
- 그래프와 비교 매물 근거가 첫 화면 안에서 더 빨리 보이도록 한다.
- 후속 피드백: 사진을 줄이면 `object-cover` 때문에 위아래가 잘릴 수 있어 `/intro` 랜딩의 2겹 이미지 패턴으로 변경했다.
- 바깥 레이어는 같은 사진을 `object-cover + blur`로 배경 처리하고, 안쪽 레이어는 `object-contain`으로 상품 사진 전체가 보이게 한다.

## 보류
- `/me/preview-detail` 별도 실험 페이지의 Hero/ProfitHero는 이번 요청 범위에서 변경하지 않았다. 실제 사용자 상세 모달을 우선 정리했다.
