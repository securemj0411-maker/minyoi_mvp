# 2026-06-04 Wave 1077 - 쉬운모드 제거와 상세 단일 화면

## 결정

- 상세 모달에서 `쉽게 보기`와 `상세보기`를 분리하지 않는다.
- 광고 유입 사용자는 두 모드 중 하나를 고르는 것보다 한 화면에서 구매 판단을 바로 보는 편이 낫다.
- 쉬운모드에 있던 핵심 신호 중 상세에서 약했던 `판매자 신뢰`는 상세 화면으로 승격한다.
- 비교매물/상태칩 노출은 기본 `detailed` 기준으로 보여준다.

## 구현

- `src/components/pack-reveal-modal.tsx`
  - 쉬운모드 자동 표시를 비활성화했다.
  - 상세 상단의 `쉽게 보기` 버튼을 제거했다.
  - `SellerTrustPanel`을 상세 상단 정보 블록 아래에 추가했다.
  - 당근은 매너온도, 중고나라는 신뢰지수/후기, 번개는 평점/후기를 판매자 신뢰 카드로 보여준다.

- `tests/detail-modal-density-contract.test.ts`
  - 쉬운모드 CTA와 자동 오버레이가 다시 노출되지 않도록 계약 테스트를 추가했다.
  - 상세에 `SellerTrustPanel`이 유지되는지 검사한다.
