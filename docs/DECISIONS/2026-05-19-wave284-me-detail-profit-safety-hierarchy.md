# 2026-05-19 Wave 284 — /me 상세 차익 요약과 안전 근거 위계 정리

## 결정
- 제품 상세 상단의 `현재 차익 / 매입 / 시세` 영역은 내부 카드처럼 보이는 테두리와 배경을 제거하고, 얇은 왼쪽 강조선이 있는 요약 정보로 낮췄다.
- `왜 이 제품이 안전한가요?` 버튼은 상품 기본 정보 카드 안에서 빼고, `왜 이걸 추천했나요?`와 같은 하단 근거 영역 위계로 분리했다.
- 안전 근거 영역은 `안전 확인` 섹션으로 두고, 추천 풀 통과 후 남은 확인 포인트를 여는 CTA를 배치했다.

## 보류
- 위험 신호 상세 모달의 문구와 축별 설명 구조는 유지했다. 이번 작업은 상세 화면의 정보 위계와 배치만 정리했다.
