# 2026-05-19 Wave 282 — /me 상세 하단 액션과 다른 매물 탐색

## 결정
- /me 추천 상세 화면은 전역 `AI 도움말` 플로팅 버튼보다 높은 레이어로 올려 상세 진입 중 오른쪽 아래 도움말 버튼이 보이지 않게 했다.
- 상세 하단 CTA에서 `공략 보기` 버튼을 제거하고, 주요 CTA는 `번개장터 열기` 하나로 정리했다.
- `거래 상태`와 `정보 오류 신고`는 고정 하단 영역이 아니라 상세 스크롤 콘텐츠의 맨 아래 액션 블록으로 이동했다.
- 상세 하단에 `/me` 목록에서 이미 들고 있는 추천 매물 캐시를 기반으로 `내 다른 추천 매물` 영역을 추가했다.

## stale 대책
- 다른 매물 카드는 `/me` 목록 캐시로 빠르게 보여주지만, 사용자가 카드를 누르면 기존 상세 열기 경로를 타면서 `onLoadDetail`이 다시 실행되어 현재 판매 상태와 상세 시세 분석을 재확인한다.
- 상세 안에 오래 머무르는 동안 판매완료될 수 있다는 안내 문구를 넣었다.

## 보류
- 상세 화면이 열려 있는 동안 하단 다른 매물 캐러셀 전체를 주기적으로 자동 갱신하는 기능은 보류했다. 지금은 사용자가 다른 매물을 열 때마다 해당 매물 단위로 재확인한다.
