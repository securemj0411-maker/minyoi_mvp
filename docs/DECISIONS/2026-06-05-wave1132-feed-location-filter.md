# 2026-06-05 Wave 1132 - Feed Location Filter

## 결정
- 추천 피드 상단 필터에 `위치` 선택을 추가했다.
- `내 동네 우선`은 저장된 사용자 동네 기준으로 당근 매물 + 거리순 서버 조회를 함께 켠다.
- `위치 전체`은 전체 출처 + 차익순으로 되돌려, 위치 할당제 메시지와 피드 조작감이 어긋나지 않게 했다.

## 보류
- 피드 안에서 임의 지역을 새로 수정하는 기능은 보류했다.
- 실제 기준 동네 변경은 온보딩/계정 설정 위치 데이터와 같이 움직여야 하므로, 별도 피드 전용 지역 상태를 만들지 않는다.
