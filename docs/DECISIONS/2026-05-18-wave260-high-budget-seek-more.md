# 2026-05-18 Wave 260 — High budget seek-more options

## 배경

사용자 피드백: `/me`의 `더 찾아보기` 예산 선택지가 최대 `80만원 이하`라서 아이패드 프로, 맥북, 고가 애플워치 같은 150만원대 상품을 스스로 제외하게 된다.

## 결정

1. 예산 선택지에 `150만원 이하`를 추가한다.
2. 예산 선택지에 `상한 없음`을 추가한다. 내부 필터는 기존 `priceMaxManwon = 0` 의미를 사용해 가격 상한을 두지 않는다.
3. 기존 `80만원 이하` 설명은 `고가 매물도 포함`에서 `태블릿/워치까지`로 낮춰 오해를 줄인다.
4. 공격형 기본 프리셋은 `80만원 이하`에서 `150만원 이하`로 상향한다.

## 보류

- 고가 매물에 추가 경고를 붙이는 것은 보류한다. 현재 이미 고수익 검색 경고가 있고, 이번 wave는 후보가 누락되는 문제를 먼저 해결한다.
