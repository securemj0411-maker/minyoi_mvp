# 2026-06-05 Wave 1118 - Apply Home Region Persist

## Decision

- 멤버십 신청 플로우의 GPS/동네 검색 결과를 `mvp_user_home_regions`에 저장하도록 연결했다.
- GPS/검색으로 찾은 동네는 `동작구 흑석동 맞나요?` 형태의 확인 박스를 보여주고, 다음 단계로 넘어갈 때 저장한다.
- 사용자가 지도에서 다른 구를 직접 고르면 GPS/검색 저장 후보는 지운다. 정확한 동네가 아닌 티오 선택을 홈지역으로 저장하지 않기 위해서다.
- 가입 완료 후 `/me`가 다시 `/onboarding/home-region`으로 보내는 문제는 이 저장 누락이 원인이었다.

## Deferred

- 지도에서 구만 직접 선택한 경우에는 아직 홈지역으로 저장하지 않는다. 당근 추천 필터는 동네 단위 정확도가 필요해서, GPS/검색으로 확인된 주소만 저장한다.
