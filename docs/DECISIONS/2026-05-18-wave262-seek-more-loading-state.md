# 2026-05-18 wave262 — 더 찾아보기 검색 진행 상태 표시

## 결정
- `더 찾아보기` 모달에서 검색 실행 중 명확한 상태 패널을 표시한다.
- 상태 패널에는 진행 문구, 설명, indeterminate progress bar, spinner를 함께 보여준다.
- 재고 확인 중에도 같은 위치에 상태를 보여 사용자가 버튼 반응 여부를 이해할 수 있게 했다.

## 보류
- 서버 단계별 실시간 스트리밍은 별도 API/SSE 설계가 필요하므로 보류했다.
- 추천 로직, 필터 로직, 크레딧 차감 로직은 변경하지 않았다.
