# Wave 249 — 모바일 추천 리포트 칩 이동 범위 보강

## 결정
- Wave 248에서 모바일 첫 카드 밖으로 뺀 `VerdictBadgesMini`에 더해, `안전/주의 ?`로 보이는 실제 위험도 칩(`RiskScoreBar`)도 `왜 이걸 추천했나요?` 안으로 함께 이동했다.
- 모바일 첫 카드의 상태 칩(`ConditionChip`)도 숨기고, 상태/시세 근거는 접힌 추천 이유 내부의 `MarketBasisMini`에서 확인하게 했다.
- 데스크톱/태블릿에서는 기존처럼 위험도, verdict, 시세 근거 칩을 카드 안에 유지한다.

## 이유
- 사용자가 말한 "원래 그냥 보이던 칩들"은 mock 요약이 아니라 실제 판단 컴포넌트 단위로 접혀야 한다.
- 작은 모바일 화면에서는 차익/매입가/시세만 남기고, 보조 판단 신호는 사용자가 의도적으로 여는 정보로 두는 편이 추천 리포트와 그래프 가독성을 동시에 올린다.

## 보류
- 원천 source badge(예: 다나와/번개 S급)는 가격 기준의 일부라 이번 wave에서는 유지했다.
