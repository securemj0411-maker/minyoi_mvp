## 결정

### 1. 헤드라인 가격 정보 단순화 (4중첩 → 2개)
- 제거: **"총차익 +N원 · +N%" 칩** — 일반인이 "예상 순익"과 차이 모름.
- 제거: **"최악 시 -X원" 배지** (wave318) — 시세 -10% 가정이 임의적이고 일반인 망설임만 키움.
- 라벨 단순화: "순익률 +N%" → "+N%" (라벨 제거, 칩 자체로 의미 충분).
- 유지: "예상 순익 +X원" / "+N%" / "판매완료 처리"(조건부).

### 2. 일수익 칩 제거
- 제거: **"일수익 표본 부족 · 회수 속도는 참고용"** — 거의 모든 매물에 떠서 가치 0.
- 사용자가 이전에도 짚었던 노이즈.

### 3. "현재성" 타일 제거 — 4 타일 → 3 타일
- 매입/시세 줄에 `9시간 전 검증` 이미 표시 → 별도 타일 중복.
- 3 타일 유지: 시장 활동(오늘 물량) / 보통 며칠에 팔림 / 거래 안전(셀러).
- Grid: 모바일 `grid-cols-1` (위→아래) + sm+ `grid-cols-3` (가로) — 모바일 가독성 우선.
- 회수 속도 라벨 변경: "회수 속도 / 보통 2일" → "보통 며칠에 팔림 / 2일" — 일반인 친화 단어.
- 표본 부족 sub 텍스트 단순화: "이 모델 표본 부족 · 카테고리 평균 N시간 기준" → "표본 적음 · 카테고리 평균 기준".

### 4. 판매 도우미 매수 전 완전 숨김

- 헤더 텍스트 단순화: "매수 후 판매 도우미" → "판매 도우미".

## 모달 정보 위계 — 정리 후

```
┌─ Layer 1 핵심 ──────────────────────────────────────┐
│  사진 + 상품명                                       │
│  예상 순익 +X원 / +N% / [판매완료 처리]              │
│  매입 X · 시세 Y · N시간 전 검증                     │
└─────────────────────────────────────────────────────┘

┌─ Layer 2 신뢰 신호 (3 타일) ─────────────────────────┐

┌─ Layer 4 위험 체크 ─────────────────────────────────┐
│  정품 확인 체크리스트 (12 카테고리만)                 │
└─────────────────────────────────────────────────────┘
```
