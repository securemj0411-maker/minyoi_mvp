# 2026-05-21 — 쉬운모드 가치 어필을 단계별로 분산

## 결정
- 쉬운모드 각 단계에 `valueNote`를 추가하고, 작은 `대신 확인한 것` 카드로 노출한다.
- 문구는 광고처럼 과장하지 않고, 사용자가 직접 하면 번거로운 일을 조용히 설명한다.
- 판매자 신뢰, 상태별 비교 매물, 시세 흐름, 판매 속도, 매입가, 리셀 비용, 채널 비교, 구매 전 체크마다 서로 다른 가치를 나눠 보여준다.

## 배경
- 쉬운모드는 첫 사용자가 상세페이지의 가치를 이해하는 핵심 진입점이다.
- 단순 요약만 보여주면 똑똑한 사용자는 가치를 알아보지만, 일반 사용자는 “왜 크레딧을 쓰는지” 체감하기 어렵다.
- 그래서 “득템잡이가 대신 정리한 노동”을 중간중간 얇게 심되, 자랑식 CTA처럼 보이지 않게 했다.

## 보류
- 오늘 차단한 매물 수, 위험 매물 수 같은 글로벌 운영 지표는 이번에는 넣지 않았다. 숫자 자랑처럼 보일 위험이 있어, 추후 별도 신뢰 배지나 서비스 안내 영역에서 다룬다.
