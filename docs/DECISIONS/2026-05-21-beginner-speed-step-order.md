# 2026-05-21 — 쉬운모드 판매 속도 단계 위치 조정

## 결정
- 판매 속도는 마지막 근거가 아니라 시세 흐름 직후에 보여준다.
- 사용자는 비교 매물과 시세 그래프를 본 뒤 바로 `되팔면 얼마나 빨리 현금화될까`를 궁금해하므로, 판매 속도를 매입가/비용 계산보다 앞에 둔다.
- 이후에 매입가, 되팔 때 비용, 안전결제, 되팔 곳 순서로 비용과 실행 판단을 이어간다.

## 구현
- 쉬운모드 순서를 `판매자 신뢰 → 비교 매물 → 시세 흐름 → 판매 속도 → 매입가 → 되팔 때 비용 → 안전결제 → 되팔 곳 → 완료`로 변경했다.
- 단계 번호를 새 순서에 맞춰 판매 속도 4번, 매입가 5번, 되팔 때 비용 6번, 안전결제 7번, 되팔 곳 8번으로 조정했다.
- 계약 테스트에 판매 속도가 시세 흐름 다음, 매입가 전에 위치한다는 회귀 조건을 추가했다.

## 보류
- 각 단계의 상세 문구와 판매 속도 계산식은 이번 변경에서 추가로 바꾸지 않았다.
