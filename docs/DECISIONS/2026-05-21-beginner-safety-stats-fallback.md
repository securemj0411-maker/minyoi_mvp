# 2026-05-21 — 쉬운모드 차단 수치 fallback

## 결정
- 쉬운모드 인트로에서 같은 상품군 차단 수치가 없으면 전체 24시간 차단 수치를 한 번 더 조회한다.
- "곧 숫자까지 보여드릴게요" 같은 placeholder 문구는 제거한다.

## 배경
- 사용자가 숫자가 보이지 않는 인트로는 가치 전달이 약하다고 지적했다.
- 특정 SKU/lane 수치가 0이어도 전체 추천 풀에서 걸러낸 24시간 수치는 서비스 가치를 보여줄 수 있다.

## 보류
- lane별 차단 수치가 자주 0으로 나오는 원인은 별도 데이터 품질 점검에서 확인한다.
