# Wave 1158 - 시세조회 크레딧 잔재 제거

## 결정
- `/lookup` 상단의 `조회 1번 = 0.2크레딧` 안내를 제거하고 멤버십 전용/추가 차감 없음 안내로 교체했다.
- `/lookup` 결과 화면의 크레딧 차감/잔액/부족 paywall UI를 제거했다.
- lookup API 주석과 푸터의 `크레딧 기반` visible 문구를 멤버십 구조에 맞게 정리했다.

## 이유
- 현재 시세조회는 이미 승인 멤버십 게이트 뒤에 있으며, 크레딧제 안내가 남아 있으면 결제 구조가 혼란스럽다.
- 사용자는 멤버십 회원인데 크레딧 차감 안내를 보면 서비스 정책이 서로 충돌한다고 느낀다.

## 보류
- 약관/환불정책/구 결제 페이지의 크레딧 정책 문구 정리는 별도 범위로 남긴다.
