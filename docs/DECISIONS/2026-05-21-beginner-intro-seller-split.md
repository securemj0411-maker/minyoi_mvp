# 2026-05-21 — 쉬운모드 인트로와 판매자 신뢰 분리

## 결정
- 쉬운모드 첫 화면을 "오늘 걸러낸 매물" 인트로로 분리했다.
- 판매자 후기/평점 안내에서 필터링 가치 안내를 제거해, 셀러 신뢰 판단과 서비스 가치 안내가 섞이지 않게 했다.
- 후기/평점 숫자 카드는 강한 초록/노랑 배경 대신 흰 배경, 얇은 링, 작은 아이콘 중심으로 낮췄다.

## 배경
- 사용자가 필터링 가치 안내는 셀러 후기 화면에 붙지 않고 독립 인트로로 보여야 한다고 지적했다.
- 기존 후기/평점 강조 카드의 색감이 촌스럽고 과하게 보였기 때문에 토스식으로 더 담백한 정보 카드에 가깝게 조정했다.

## 보류
- 인트로 문구와 breakdown 사유 명칭은 실제 데이터 분포를 더 본 뒤 계속 조정한다.
