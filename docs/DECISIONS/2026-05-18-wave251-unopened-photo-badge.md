# Wave 251 — 미개봉/새상품 사진 뱃지

## 결정
- `conditionClass === "unopened"` 매물은 `/me` 상품 카드와 추천 리포트 모달의 사진 위에 미개봉 뱃지를 노출한다.
- `/me` 작은 썸네일에서는 `미개봉`으로 짧게 표시하고, 모달의 큰 이미지에서는 넓은 화면에서 `미개봉/새상품`까지 확장한다.
- 기존 상태 칩(`ConditionChip`)은 가격 기준 설명용으로 유지하고, 사진 뱃지는 첫눈에 상태를 알아보는 시각 신호로 분리했다.

## 이유
- 미개봉/새상품은 차익 판단에서 매우 큰 상태 신호라 텍스트 줄 안에 묻히면 안 된다.
- 사용자는 이미지를 먼저 보므로 사진 위 뱃지가 가장 빠르게 "이 매물은 새상품 기준으로 계산됐다"는 맥락을 준다.

## 보류
- `S급`, `A급`, `사용감` 등 다른 상태의 사진 뱃지는 아직 추가하지 않았다. 이번 wave는 사용자가 요청한 미개봉/새상품만 우선 적용했다.
