# 2026-06-05 Wave 1102 — 피드 업셀 입금 확인 카운트다운 표시

## 결정

- 피드 업셀 모달에서 `입금했어요` 버튼을 누른 뒤 5분 자동승인 카운트다운을 표시하도록 수정했다.
- `/api/membership/deposit-notify`가 내려주는 `scheduledAutoApproveAt`을 우선 사용하고, 응답에 값이 없을 때는 클라이언트 기준 5분 fallback을 사용한다.
- 사용자가 기다리는 동안 `자동 승인 대기 중`과 남은 시간을 볼 수 있게 했다.

## 보류

- 자동승인 후 피드 상태를 즉시 갱신하는 별도 polling UX는 이번 작업에 포함하지 않았다. 이번 수정은 입금 요청 이후 불안감을 줄이는 카운트다운 표시가 목적이다.
