# 2026-05-19 Wave 303 — /me terminal summary placement

## Context

사용자가 `/me` 상단에 보이는 "판매완료된 상품 N건을 기록으로 남겨뒀어요" 배너가 첫 화면을 차지해 불필요하게 느껴진다고 지적했다.

해당 배너는 판매완료/시세 마감 기록을 보존하고 접는 기능을 제공하려는 의도였지만, 상품 피드와 CTA보다 먼저 노출되어 신규/재방문 사용자의 주의를 빼앗았다.

## Decisions

1. 상품 목록 위 terminal 안내 배너를 제거했다.
   - "기록으로 남겨뒀어요" 같은 긴 설명 문구는 첫 화면에서 빠졌다.
   - 주황색 강조 버튼도 제거해 주요 CTA처럼 보이지 않게 했다.

2. 판매완료 기록 접기/보기 컨트롤은 상품 목록 아래로 내렸다.
   - 보조 정보 성격에 맞게 얇은 border-top + 작은 텍스트 행으로 변경했다.
   - 모바일에서는 `판매완료 기록 N건 포함/숨김` 정도만 보이고, 시세 마감 세부 숫자는 데스크톱에서만 보인다.
