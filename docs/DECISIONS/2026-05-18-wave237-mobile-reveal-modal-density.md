# 2026-05-18 Wave 237 — 상품 보기 모바일 정보 밀도 재정렬

## 배경

사용자 요청: 상품 보기 모달에서 상단 매물 카드의 시세 근거/주의 chip 정보가 너무 길어, 모바일에서 아래 시세 그래프와 회전 분석이 한눈에 들어오지 않는다. 또한 `왜 이걸 추천했나요?`는 사진 카드와 그래프 사이에 배치하고, 셀러 후기·판매 속도·주의점까지 더 예쁘고 쉽게 설명해야 한다.

## 결정

- 모바일 읽기 순서는 `매물 핵심 → 왜 추천했는지 → 시세 그래프`로 둔다.
- 데스크톱은 기존처럼 좌측 매물 카드와 우측 그래프를 먼저 나란히 보여주고, 추천 이유는 아래 full-width로 둔다.
- `시세 근거`는 기본 상태에서 한 줄 요약으로 축소한다.
  - 상태 등급
  - 다나와/번개 기준
  - 표본 수
  - 신뢰도
- 상세 모델명, 거래완료/판매중 수, 다른 등급 시세는 `자세히`를 눌렀을 때만 보여준다.
- 카드의 verdict chip은 모바일에서 앞 3개만 노출하고 나머지는 `+N`으로 접는다. 상세 설명은 추천 이유 패널로 넘긴다.
- 추천 이유 패널은 좋은 점과 확인할 점을 분리한다.
  - 좋은 점: 현재 차익, 셀러 후기, 판매 속도, 다나와/상태별 시세 기준, 무료배송 등
  - 확인할 점: 시세 표본 낮음, 사용감은 같은 등급 시세에 반영됨, 기존 warning verdict

## 반영

- `MarketBasisMini` 기본 UI를 compact summary로 변경했다.
- `RecommendationReasonPanel`에 `좋은 점 / 확인할 점` 영역을 추가했다.
- `RecommendationReasonPanel`을 모바일에서는 매물 카드 바로 아래, 데스크톱에서는 그래프와 나란히 본 뒤 아래에 보이도록 order class를 적용했다.
- `RiskScoreBar`는 상품 보기 모달에서 compact 모드로 노출한다.
- `VerdictBadgesMini`는 모바일에서 3개 초과 chip을 접는다.
- `/me` contract test에 모바일 정보 밀도와 추천 이유 설명 회귀 검증을 추가했다.
