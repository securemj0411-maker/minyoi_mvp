# 2026-05-21 Explore category controls and buy-price sort

## Context
- `/me` 추천 피드의 카테고리 칩이 가로 스크롤만 지원해 PC 사용자가 트랙패드 없이 넘기기 어렵다.
- 리셀러는 차익뿐 아니라 낮은 매입 단가부터 훑는 흐름이 필요하다.

## Decision
- 카테고리 칩 영역에 PC용 좌우 이동 버튼을 추가한다. 모바일 터치 스와이프는 유지한다.
- 정렬 옵션에 `매입단가순`을 추가하고, 서버 응답에서 최종 표시 매물을 매입가 낮은순으로 정렬한다.

## Deferred
- 고가/저가 범위 슬라이더, 카테고리별 서버 필터링, 매입가 구간 프리셋은 후속 작업으로 둔다.
