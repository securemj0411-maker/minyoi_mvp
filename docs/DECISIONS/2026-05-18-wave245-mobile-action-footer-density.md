# 2026-05-18 Wave 245 — /me 모바일 하단 액션바 압축

## 배경

사용자 지적: `/me` 상품 모달에서 하단 고정 액션 영역이 모바일 화면을 너무 크게 차지해 `시세 그래프 · 시장 분석` 그래프가 한눈에 보이지 않는다.

Wave 241/242에서 거래 상태, 매수 후 진행, 오류 신고 루프를 추가했지만, 모바일에서는 retention control이 시각 분석 영역을 밀어내는 부작용이 생겼다.

## 결정

- 모바일에서는 하단 액션바를 2단 compact layout으로 줄인다.
  - 1단: `공략 보기`, `번개장터 열기`
  - 2단: 작은 `거래 상태` summary, 작은 `오류 신고 +3`
- 거래 상태 3버튼과 매수 후 진행 버튼은 기본 노출하지 않고, `거래 상태`를 눌렀을 때 하단 바 위로 뜨는 접힘 패널 안에 둔다.
- `sm` 이상 화면에서는 기존 넓은 거래 상태 rail을 유지한다.

## 보류

- footer를 완전히 sticky 해제하는 안은 보류한다. 핵심 CTA 접근성은 유지하되 모바일 기본 높이만 줄인다.
- 거래 상태 패널을 별도 bottom sheet로 분리하는 작업은 보류한다.

## 검증

- `/me` contract test에 모바일 compact footer 구조(`sm:hidden`, `hidden sm:block`, 상태 패널 upward popover, `오류 신고 +3`)를 추가한다.
