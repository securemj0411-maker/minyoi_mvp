# 2026-06-05 Wave 1112 - Plans Region Map Drilldown

## 결정

- 멤버십 신청 첫 화면의 지역 지도는 초기 상태에서 광역시/도 단위로만 읽히게 한다.
- 시/군/구 단위 경계와 대표 지역 텍스트는 사용자가 광역 지역을 선택한 뒤 확대 상태에서만 노출한다.
- 모바일 하단의 `수원시`, `성남시` 같은 지역 칩 목록은 제거한다. 좁은 화면에서는 지도와 남은 티오가 먼저 보여야 한다.

## 구현

- `PlansApplicationFlow`에 `mapZoomed` 상태를 추가해 전국 지도 상태와 선택 지역 상세 상태를 분리했다.
- `KoreaSeatMap`은 초기에는 내부 경계선을 지역 색으로 묻고, 선택된 광역 지역에만 상세 경계선을 다시 켠다.
- 지역을 클릭하면 해당 광역 지역 중심으로 지도를 확대하고, 선택 지역의 남은 티오를 크게 표시한다.
- 데스크톱 우측 패널도 초기에는 전체 남은 자리/예약률을 보여주고, 선택 후에는 선택 지역 기준으로 바뀐다.
