# 2026-05-21 Beginner conversion and evidence pass

## Context
- 손쉬운 모드가 초반 3건 무료에서 가치를 빠르게 심는 역할이라면, 설명을 줄이고 행동/전환 근거를 더 보여줘야 한다는 피드백을 반영했다.

## Decisions
- 손쉬운 모드의 `매입가`와 `되팔 때 비용` 2장을 `최종 순익` 1장으로 압축했다.
  - 상품가, 구매 배송비, 수익 기준 시세, 되팔 때 수수료/재배송/버퍼를 한 화면에 묶어 스크롤 부담을 줄였다.
- 무료 3건 이후 크레딧 부족 모달에 이번 세션에서 받은 가치 요약을 추가했다.
  - 비교 매물 수, 예상 기회 수익, 주의 신호, 절약한 판단 시간을 보여줘 단순 결제 요구가 아니라 "무엇을 받았는지"가 남게 했다.
- 상세 숫자 리포트의 비교 매물에는 원문 보기와 근거 태그를 추가했다.
  - 상세 모드에서 `동일 기준`, `상태 유사`, `참고 매물`, 판매 상태, 마지막 확인 시점을 함께 보여준다.

## Deferred
- 무료 3건을 수익형/위험 회피형/애매한 판단형으로 의도 배치하는 것은 추천 풀 selection 정책이 필요해 후속 작업으로 둔다.
- 비교 매물의 동일/유사 모델 판정은 현재 endpoint의 세부 유사도 필드가 부족해 우선 UI 태그 수준으로 시작한다.
- paywall 가치 요약은 현재 세션 기반이다. 계정 간/새로고침 이후 누적 요약은 별도 저장소 설계 후 진행한다.
