# 2026-05-22 Wave 532 — `/me` 모바일 피드 필터 압축

## 결정
- 모바일 `/me` 피드에서 상단 설명 배너(`같은 상태 매물끼리만 비교...`)는 숨긴다.
- 같은 내용은 상세/신뢰 근거에서 충분히 반복되므로, 모바일 첫 피드에서는 매물과 필터 조작성을 우선한다.
- 필터 바는 한 줄에 모두 욱여넣지 않고, 모바일에서 2단 구조로 바꾼다.
  - 1단: 스크랩 + 카테고리 가로 스크롤
  - 2단: 예산 / 출처 / 정렬 3등분 select
- 데스크탑은 기존 배너와 한 줄 필터 구조를 유지한다.

## 보류
- 카테고리 아이콘 버튼을 눌러 bottom sheet로 여는 방식은 이번에는 보류한다. 현재는 가로 스크롤 칩을 유지한다.
