# 2026-05-19 wave302 - 상세 지표 텍스트 클리핑 방지

## 결정
- /me 상세 모바일 상단 지표 그리드의 좌우 확장을 `-mx-3`에서 `-mx-[10px]`로 줄였다.
- 지표 카드 사이 간격을 `gap-px`에서 `gap-0.5`로 늘려 테두리와 텍스트가 화면 끝에서 잘리지 않게 했다.
- 카드 내부 좌우 패딩을 `px-2`에서 `px-2.5`로 늘려 `보통`, `1시간`, `근거 보기` 같은 짧은 텍스트도 좌우가 잘리지 않게 했다.

## 보류
- 2x2 지표 구조는 유지했다. 4열 한 줄은 더 많은 텍스트 절단을 만들 수 있다.
