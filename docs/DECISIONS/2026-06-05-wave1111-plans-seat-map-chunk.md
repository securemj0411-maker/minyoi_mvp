# 2026-06-05 Wave 1111 - Plans 첫 화면 지역 티오 지도 chunk

## 결정
- `/plans` 비회원 첫 화면에 대한민국 남한 형태의 지역 티오 SVG chunk를 추가했다.
- 지역을 누르면 해당 권역이 강조되고, 구/시 단위 남은 티오와 과밀 상태를 보여주도록 했다.
- 기존 신청 카드 안의 선착순 티오 bar는 제거해 전체 예약 수가 두 번 크게 반복되는 문제를 줄였다.
- 모바일 fixed 신청 CTA와 고객센터 FAB가 겹치지 않도록 고객센터 FAB의 모바일 위치를 위로 올렸다.

## 보류
- 실제 지역별 신청자 DB 집계 연동은 보류했다. 현재는 선공개 멤버십 전환용 mock 티오 데이터로 UI 위계를 먼저 정리했다.
- GIS 수준의 정밀 지도 대신 가입 페이지용 경량 SVG 지도로 처리했다.
