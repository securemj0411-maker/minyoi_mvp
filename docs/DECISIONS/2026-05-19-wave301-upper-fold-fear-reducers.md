# 2026-05-19 Wave 301 — 상세 상단 fear reducer 재배치

## 배경
- 모바일 상세 상단에서 사용자가 가장 먼저 느끼는 불안은 “돈이 묶일까”, “안전한 매물인가”, “진짜 시세보다 싼가”, “비슷한 매물 표본이 충분한가”다.
- 기존 판매 속도 카드는 `2일 안에 팔렸어요` 숫자가 크게 보여서 메시지는 강했지만, 상단 공간을 너무 많이 차지했다.

## 결정
- 상세 상단 가격 블록 바로 아래에 2x2 compact 타일을 배치했다.
- 타일은 `회수 속도`, `거래 안전`, `시세 확신`, `매물대` 네 가지로 고정한다.
- `2일` 같은 판매 속도 값은 큰 히어로 숫자가 아니라 14~15px 수준의 판단 신호로 낮췄다.
- “왜 이걸 추천했나요?” 패널은 그 아래에 유지해서, 상단은 불안 해소 요약 / 하단은 설명 구조로 분리한다.

## 보류
- 판매 속도 그래프와 상세 시세 근거의 정보량 자체는 이번 변경에서 더 줄이지 않았다.
- 추후 모바일 첫 화면 캡처를 보면서 이미지 높이, 가격 카드, 2x2 타일, 추천 이유 패널의 실제 fold 점유율을 다시 조정한다.
