# 2026-05-21 Free Feed Preview Lock

## 결정
- 무료 피드는 원제목, 원본 사진, 정확한 가격을 바로 노출하지 않고 모델군/상태/가격대 중심의 미리보기로 제한한다.
- 상세 분석 접근권한을 얻은 매물은 같은 세션에서 원제목과 사진을 다시 노출한다.
- 크레딧 보유자는 기존처럼 피드를 넓게 탐색할 수 있게 두고, 상세 분석은 기존 차감 정책을 유지한다.

## 이유
- 무료 피드가 엄선된 매물의 원제목과 정확한 가격을 모두 보여주면, 사용자가 번개장터에서 직접 검색해 상세 분석을 우회할 수 있다.
- 무료 경험은 "기회가 있다"를 체감시키고, 실제 구매 행동에 필요한 원본 매물/판매자/정확한 수익 정보는 상세 분석에서 제공하는 구조가 유료 전환에 더 맞다.

## 보류
- 서버에서 이미 열람한 pid 목록을 피드 응답에 포함해 새로고침 후에도 즉시 해제하는 기능은 보류한다. 현재는 세션/스크랩 기반으로 우선 처리한다.
- 무료 피드 잠금 강도를 A/B 테스트하는 정책은 추후 데이터가 쌓이면 다시 결정한다.
