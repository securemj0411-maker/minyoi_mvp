# Wave 1065 — Detail sale speed emphasis

## Decision

- 상세보기의 판매속도 표현을 `빠름/보통/느림` 판정 중심에서 `평균 N일 내로 팔려요` 문장 중심으로 바꿨다.
- 판매속도 요약 타일을 예상 순익 카드 바로 아래로 올렸다.
  - 기존 순서: 예상 순익 → 그래프 → 비교매물 → 왜 싼지 → 판매속도/거래안전
  - 변경 순서: 예상 순익 → 판매속도/거래안전 요약 → 그래프 → 비교매물
- 쉬운모드 판매속도 카드도 같은 문장 톤으로 맞췄다.
- 상세 상단의 중복 메타를 줄였다.
  - 당근 동네는 아래 `거래 가능 동네` chip으로만 보여준다.
  - `AI 판단 · 매물 설명(텍스트) 기준 · 사진은 직접 확인 권장` 문구는 상단에서 제거했다.
  - 상단의 `득템 점수` 미니 점수는 구매 판단 요약/근거 영역과 중복되어 제거했다.

## Deferred

- 판매속도 요약을 단독 full-width 카드로 더 크게 분리할지는 실제 모바일 화면 확인 후 결정한다.
