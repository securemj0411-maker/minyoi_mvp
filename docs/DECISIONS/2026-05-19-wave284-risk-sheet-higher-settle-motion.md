# 2026-05-19 Wave 284 — 위험 설명 패널을 더 위로 올리고 짧은 정착 모션 적용

## 결정
- 위험 신호 설명 패널의 모바일 위치를 화면 정중앙보다 살짝 위(`top-[45%]`)로 조정했다.
- 하단 고정 CTA와 겹치지 않도록 패널과 내부 스크롤 영역의 최대 높이를 줄였다.
- 아래에서 길게 올라오는 바텀시트 모션은 사용하지 않고, 중앙 위치에서 6px 정도만 위로 정착하는 `fade + settle` 모션을 적용했다.

## 이유
- 사용자가 누른 설명은 바텀 액션이 아니라 안전 신뢰 설명이므로 아래에서 올라오면 바텀시트처럼 오해된다.
- 중앙보다 약간 위에 두면 사진/카드 위에 안정적으로 떠 보이고, 하단 CTA도 가리지 않는다.

## 보류
- 트리거 버튼 위치에서 확장되는 shared-element 스타일 모션은 보류했다.
- 실제 기기별 safe-area 세부 튜닝은 배포 후 스크린샷을 보고 추가 판단한다.
