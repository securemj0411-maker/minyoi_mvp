# 2026-05-21 — PC 상세보기 모바일 셸 고정

## 결정
- PC 상세보기는 넓은 2단 분석 모달을 쓰지 않고, 모바일 상세 화면을 데스크톱 배경 위에 띄운 형태로 고정한다.
- 데스크톱 뷰포트에서도 내부 콘텐츠는 480px 안팎의 단일 컬럼 흐름을 따른다.
- 로딩 스켈레톤과 하단 거래 상태 컨트롤도 PC에서 모바일과 같은 밀도로 보이게 한다.

## 구현
- `PackRevealModal`의 내부 셸에 `data-mobile-detail-shell`을 추가하고, `sm:max-w-6xl` 기반의 와이드 모달을 `sm:max-w-[480px]` 셸로 교체했다.
- 상세 카드의 `lg:grid-cols-2`, 사진/본문 데스크톱 2열, 데스크톱 전용 요약 블록을 제거했다.
- 로딩 스켈레톤과 득템 점수 버튼의 데스크톱 전용 축소/2열 클래스를 제거했다.
- 계약 테스트에 PC에서도 모바일 상세 셸을 유지한다는 회귀 조건을 추가했다.

## 보류
- 초보자 가이드, 추천 이유 팝업, 관련 매물 사이드 프리뷰의 세부 UX는 이번 변경에서 새로 설계하지 않았다.
- 필요하면 다음 작업에서 모든 보조 패널까지 480px 셸 안에 강제로 맞추는 2차 정리를 진행한다.
