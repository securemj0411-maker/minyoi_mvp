# 2026-05-21 — 쉬운모드 인트로 수치 로딩 마스킹

## 결정
- 쉬운모드 첫 인트로에서 안전 수치 API를 기다리는 동안 바로 "확인 중"을 보여주지 않는다.
- 처음 약 760ms 동안 "추천 풀을 훑는 중" 로딩 카드와 pulse row를 보여준다.
- 그 사이 수치가 도착하면 바로 수치 카드로 전환하고, 늦으면 기존 확인 중 상태로 넘어간다.

## 배경
- 인트로 숫자가 늦게 나타나면 사용자가 계산이 오래 걸린다고 느낀다.
- 스켈레톤만 보여주는 것보다 짧은 작업 애니메이션으로 "검사 중" 맥락을 주는 편이 자연스럽다.

## 보류
