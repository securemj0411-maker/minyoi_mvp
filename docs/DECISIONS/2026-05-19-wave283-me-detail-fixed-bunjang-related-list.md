# 2026-05-19 Wave 283 — /me 상세 번개장터 CTA 고정과 관련 매물 리스트형 전환

## 결정
- `/me` 상세 화면에서 `번개장터에서 확인하기`는 하단 고정 CTA로 유지한다.
- `거래 상태`와 `정보 오류 신고`는 스크롤 콘텐츠 맨 아래에 남겨 상세 정보와 함께 정리한다.
- `내 다른 추천 매물`은 2열 카드 그리드가 아니라 당근마켓형 1열 리스트로 바꿨다. 왼쪽에는 사진, 오른쪽에는 상품명·예상 차익·상태 재확인 안내를 둔다.

## 보류
- 관련 매물 리스트의 정렬/추천 기준 고도화는 보류했다. 현재는 `/me`가 이미 들고 있는 같은 페이지 캐시에서 현재 상품과 마감 상품을 제외한 순서대로 보여준다.
