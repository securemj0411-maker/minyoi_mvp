# 2026-05-21 — 쉬운모드 구매 전 체크 SKU 위험도 게이트

## 결정
- 쉬운모드 배터리 체크에서 `earphone` 카테고리를 제외한다. AirPods/AirPods Max처럼 배터리 효율 화면이 없는 제품에 배터리 효율 확인처럼 보이는 안내를 띄우지 않는다.
- 쉬운모드 정품 확인 체크는 카테고리 전체가 아니라 SKU/브랜드 위험도 기준으로 노출한다.
- AirPods Max는 `counterfeitRisk: low`로 내리고, 정품/가품 안내보다 기능·구성품·보증 상태 확인을 우선한다.

## 배경
- AirPods Max 상세 쉬운모드에서 “배터리 상태를 물어보세요”, “정품 확인 포인트”가 동시에 떠서 실제 사용자 관점에서 과잉 경고처럼 보였다.
- AirPods Pro/일반 AirPods처럼 가품 위험이 큰 lane과 AirPods Max 같은 lane을 같은 `earphone` 규칙으로 묶으면 신뢰도가 떨어진다.

## 보류
- 상세 분석의 카테고리 체크리스트 전체를 SKU별 커스텀 리스트로 완전히 분리하는 작업은 별도 UX 정리로 둔다.
