# 2026-05-21 Beginner guide action CTA follow-up

## Context
- 두 개의 외부/사용자 피드백을 합쳐 손쉬운 모드를 다시 점검했다.
- 전제는 "상세 숫자 리포트는 따로 있고, 손쉬운 모드는 초반 3건 무료에서 빠르게 가치를 심는 화면"이다.

## 적극 수용
- `초보자 가이드 스킵하기` 문구는 하급 모드처럼 보이고 부가 설명처럼 느껴져 `상세 숫자 리포트 보기`로 바꿨다.
- 마지막 화면은 단순 완료 아이콘 대신 최종 판단, 예상 순익, 최대 매입가, 비교 매물 수, 확인 질문 수를 보여주는 행동 요약으로 바꿨다.
- 구매 전 체크에는 바로 쓸 수 있는 `문의 문구 복사` 버튼을 추가했다.
- 판매처 비교에는 `추천: 당근 먼저 등록` 또는 `추천: 번개장터 먼저 등록` 문장을 붙여 숫자 비교가 행동으로 이어지게 했다.

## 부분 수용
- "처음부터 최종 요약을 1장으로"는 강한 전환 장치지만, 현재 이미 첫 장이 필터링 가치와 연결되어 있어 이번 변경에서는 마지막 판단/행동을 먼저 강화했다.
- "6/7장 비용 계산 합치기"는 방향은 맞지만, 기존 비용 카드와 테스트 범위가 넓어 별도 UX 패스로 분리한다.
- "판매 속도에 카테고리 평균 대비 빠름/느림"은 좋은 판단 기준이나 현재 데이터 모델에 카테고리 평균 판매 기간이 안정적으로 노출되어 있지 않아 보류했다.

## 보류
- 무료 3건을 수익형/위험 회피형/애매한 매물 판단형으로 의도 배치하는 것은 추천 풀과 paywall 상태 설계가 필요해 별도 작업으로 둔다.
- 유료 전환 직전 누적 가치 요약, 저장/공유, 판매글 자동 작성, 사진 AI 분석은 결제 전환에는 유효하지만 이번 상세 모드 정리 범위를 넘겨 후속 작업으로 남긴다.
