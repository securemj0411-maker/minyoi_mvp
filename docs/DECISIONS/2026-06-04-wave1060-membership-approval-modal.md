# 2026-06-04 Wave 1060 — membership approval modal

## Context
- 멤버십 승인 감지 후 기존 UI는 화면 상단의 작은 알림과 짧은 자동 이동이었다.
- 사용자가 기다리다가 다른 작업을 하면 작은 알림을 놓칠 수 있고, 입금/승인 전환의 중요도에 비해 피드백이 약했다.

## Decision
- 승인 감지 시 상단 토스트 대신 화면 중앙 modal dialog를 띄운다.
- 자동 리다이렉트는 제거하고, 사용자가 `상품 피드로 이동` 버튼을 눌렀을 때 `/me`로 이동한다.
- 신규 가입, 연장 승인, 이미 활성화된 계정 모두 같은 modal path를 쓴다.

## Deferred
- 승인 modal에서 남은 멤버십 기간/만료일을 같이 보여주는 확장은 보류한다.
