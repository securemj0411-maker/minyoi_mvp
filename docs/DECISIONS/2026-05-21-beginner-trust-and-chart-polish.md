# 2026-05-21 — 쉬운모드 신뢰 기준과 시세 그래프 polish

## 결정
- 판매자 신뢰 문구는 후기 10건 이상부터 `신뢰가 있는 판매자`로 말한다.
- 후기가 10건 미만이면 평점이 높아도 `평점은 좋지만 판단 표본이 적다`고 안내한다.
- 비교 매물 문구는 `낮아요`보다 사용자가 바로 이해하는 `싸게 나왔어요`를 쓴다.
- 쉬운모드 시세 그래프는 선이 그려지는 가벼운 SVG 애니메이션을 적용하되, reduced-motion 환경에서는 꺼진다.

## 구현
- `SELLER_TRUST_MIN_REVIEW_COUNT = 10` 기준을 추가했다.
- `BeginnerGuideTrustBody`가 후기 수 부족 케이스를 별도 문장으로 렌더링하게 수정했다.
- 비교 매물의 할인 문구를 `상태가 비슷한 매물보다 싸게 나왔어요`, `N원 저렴`으로 바꿨다.
- `MarketHistoryChart`에 line draw, 가격선 scale, 점 pop 애니메이션을 추가했다.
- 계약 테스트에 후기 기준, 카피 회귀, 그래프 애니메이션 회귀 조건을 추가했다.
