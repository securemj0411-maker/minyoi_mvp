# 2026-05-21 Remove Explore Budget Gate

## 결정
- `/me` 추천 피드의 가입 직후 예산 질문을 제거한다.
- 무료 사용자의 더 찾아보기 모달에서 `제한 없음 · 균형 · 수정` 선호 요약을 제거한다.
- pool API 호출에는 예산/성향 파라미터를 보내지 않고, 기본적으로 전체 후보를 최대한 넓게 보여준다.

## 이유
- 현재 후보 pool은 카테고리와 상태별로 희귀한 편이라, 예산 필터를 먼저 적용하면 사용자가 매물이 적다고 느끼기 쉽다.
- 초반 경험에서는 개인화보다 "볼 만한 후보가 계속 있다"는 밀도가 더 중요하다.

## 보류
- 사용자가 명시적으로 가격대 필터를 선택하는 고급 필터는 추후 별도 UI로 다시 검토한다.
- 기존 로컬스토리지에 저장된 예산/성향 값은 삭제하지 않는다. 다만 현재 피드 호출에는 사용하지 않는다.
