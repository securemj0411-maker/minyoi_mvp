# 2026-06-04 Wave 1099 — 피드 상단 보조 proof strip 제거

## 결정

- `/me` 추천 피드 상단의 보조 proof strip을 제거했다.
- 제거 문구: `같은 상태 매물끼리만 비교 — 진짜 싼 매물만`, `오늘 N건 잡힘`, `대기 없이 즉시 받기`.
- 멤버십 신청제 전환 이후 `대기 없이 즉시 받기`는 승인/선착순 포지셔닝과 충돌하고, 같은 상태 비교 설명은 상세 근거 영역에서 충분히 전달하는 편이 낫다고 판단했다.

## 보류

- `/api/stats/pool` 기반 통계 호출은 아직 다른 empty/exhausted 상태 안내에서 사용 중이라 유지했다.
- 같은 상태 비교 원칙 자체는 제거하지 않았다. UI 상단 반복 노출만 줄이고, 상세 근거/비교 매물 영역에서 맥락에 맞게 보여준다.
