# Wave 1164 — 멤버십 입금 모달 위계/송금 선택지 정리

- 결정: 멤버십 신규/연장 입금 모달을 header보다 위에 뜨도록 z-index를 올리고, 모달 자체가 화면 안에서 시작되도록 overlay 스크롤 구조로 바꿨다.
- 결정: 큰 제목의 `3개월 연장 예약을 잡았어요` 같은 중복 문구를 제거하고 `입금 방법 선택` / `입금 확인 중` 중심으로 위계를 낮췄다.
- 결정: 송금 방식은 `토스로 송금하기`와 `계좌로 직접 송금하기`를 `또는` 구분선으로 나눠, 둘 중 하나를 선택하는 구조로 명확하게 바꿨다.
- 결정: 피드 업셀 결제 영역도 같은 송금 선택지 톤으로 맞췄다.
- 검증: `npm run lint` 에러 0개(기존 warning 유지), `npm run build` 성공.
- 보류: 실제 로그인 세션이 필요한 운영자/멤버십 화면의 브라우저 실측은 배포 후 운영 계정으로 한 번 더 확인한다.
