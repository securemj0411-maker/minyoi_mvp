# 2026-05-19 Wave 302 — 상세 상단 trust check 재구성

## 배경
- 사용자가 제안한 `돈 묶임`, `안전`, `매물대`, `시세보다 싼지`는 방향은 맞지만, 그대로 네 칸으로 두면 설명이 기계적으로 보였다.
- 모바일 상세 첫 화면에서는 상품 사진/가격/CTA 때문에 하단 근거를 보지 않는 사용자가 많다.

## 결정
- 상세 상단의 compact 신호를 `현재성`, `시세/매물대`, `회수 속도`, `거래 안전`으로 재정렬했다.
- `현재성`은 “이 상품 아직 살아 있나?” fear를 줄이기 위해 마지막 검증 시간을 첫 칸으로 올렸다.
- `시세/매물대`는 할인율만 보여주지 않고 `상태 기준 + 출처 + 표본 수`를 한 줄에 붙여서 “같은 상태끼리 비교했나?”를 바로 알 수 있게 했다.
- `회수 속도`는 큰 hero 숫자 대신 작은 판단 신호로 유지했다.
- 타일 색은 Toss식으로 면 전체를 강하게 칠하지 않고, 흰 배경 + 작은 색 점 + value 색상만 사용한다.

## 보류
- 실제 authenticated `/me` 상세 데이터의 first fold 캡처 기준으로 이미지 높이와 추천 이유 카드 위치는 다음 UI 패스에서 다시 다듬는다.
- 판매 속도 fallback 2일은 UI 테스트용으로 유지하되, 데이터가 쌓이면 실측값 중심으로 전환한다.
