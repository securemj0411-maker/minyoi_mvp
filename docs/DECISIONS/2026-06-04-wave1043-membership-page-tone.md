# 2026-06-04 Wave 1043 — membership application page tone

## Trigger

멤버십 신청 페이지가 `결제 페이지가 아니라 신청 페이지예요`라는 방어적 H1로 시작해 premium/application tone이 약했다.

## Decisions

- H1을 `선공개 300명 멤버십`으로 바꾸고, `신청제 · 운영자 검토 · 승인 후 활성`을 보조 badge로 둔다.
- 가격은 숨기지 않는다.
  - 10만원 일시불은 부담이 있지만, 실제 체감 단가인 `월 33,000원꼴`을 먼저 보여준다.
  - `3개월 99,000원 · 승인 후 결제 안내`는 보조 문구로 둔다.
- hero copy는 방어/설명 톤 대신 가치와 희소성을 먼저 말한다.
  - 돈 되는 매물이 극소수
  - 아무나 보면 사라짐
  - 신청/검토/승인된 분만 봄

## Deferred

- 신청 폼의 세부 질문 입력 UI는 아직 만들지 않았다. 현재는 카카오 로그인 후 즉시 신청 row를 남기는 구조를 유지한다.
