# 2026-05-20 Related Item Paywall Scroll

## Context

상세 모달 하단 `다른 수익 매물`에서 새 매물을 누르면 상세보기 접근 권한을 먼저 확인한다.

기존 구현은 권한 확인 전에 모달 스크롤을 맨 위로 올렸다. 그래서 무료 상세보기 3회를 모두 쓴 사용자는 현재 위치를 잃은 뒤 제한 모달을 보게 됐다.

## Decision

- 관련 매물 클릭 직전 강제 스크롤 콜백을 제거한다.
- 스크롤 초기화는 실제로 열린 매물의 `activeRevealPid`가 바뀔 때만 수행한다.
- 접근 제한/paywall이 뜨는 경우에는 기존 상세 모달 위치를 유지한다.

## Deferred

- paywall 모달이 열린 위치의 하단 CTA와 겹치는지 여부는 실제 사용자 화면에서 추가 확인한다.
