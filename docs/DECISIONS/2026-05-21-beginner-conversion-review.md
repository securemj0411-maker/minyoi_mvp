# 2026-05-21 — 쉬운모드 전환율 피드백 반영

## 결정
- 첫 필터링 화면에 `후보 N건 중` 모수를 추가했다.
- 판매자 신뢰 표현을 `신뢰가 있는 판매자`에서 `신뢰 신호가 있는 편`으로 톤다운했다.
- 비교 매물 화면의 표본 문구를 `총 N건 중 비싼 순 일부`로 바꿨다.
- 쉬운모드에서는 2일치 그래프 단계가 흐름을 끊지 않도록 시세 흐름 step을 제외했다.
- 강한 전환 포인트인 비교 매물과 판매처 비교를 앞쪽으로 이동했다.

## 배경
- 사용자는 유료 전환에는 `얼마나 싸게 나왔는지`, `어디에 팔면 얼마가 남는지`가 초반에 보여야 한다고 피드백했다.
- 후기/평점만으로 판매자 신뢰를 단정하는 것은 위험하므로 표현을 보수적으로 낮췄다.
