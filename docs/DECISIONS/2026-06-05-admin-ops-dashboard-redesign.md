# 2026-06-05 Admin Ops Dashboard Redesign

## 결정
- cau 운영자 루트를 멤버십 입금 확인, 고객상담, 회원 관리, 오류 신고 섹션으로 분리했다.
- 멤버십 신청 목록은 `입금 전 예약`, `입금 확인 요청`, `최근 처리 기록`으로 나눠 보여준다.
- 관리자 페이지 진입 시 오래된 미입금 예약은 7분 만료로 정리하고, 5분이 지난 입금 확인 요청은 cron 누락 시 자동승인 보정되게 했다.
- 고객상담은 운영자 답장 시 사용자 고객센터 버튼에 안 읽은 배지가 뜨도록 status/read API를 추가했다.
- Bloomberg terminal 톤의 영문 라벨을 한국어 운영 대시보드 톤으로 바꿨다.

## 보류
- 회원 상세에서 특정 고객에게 바로 새 상담을 여는 기능은 기존 상담 스레드가 있는 경우 연결만 제공했다. 신규 outbound 상담 생성은 별도 운영 정책 확정 후 구현한다.
- 관리자 화면을 `/payments`, `/support`, `/members` 별도 라우트로 완전 분리하는 작업은 이번에는 하지 않고 루트 내 섹션 분리로 먼저 정리했다.
