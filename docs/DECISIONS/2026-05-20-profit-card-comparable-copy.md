# 2026-05-20 Profit Card Comparable Copy

## Context

상세 모달의 예상 순익 카드 안 버튼이 `계산식 · 비교 매물 0개 보기`라고 표시됐다.

하지만 비교 매물은 예상 순익 카드 바로 아래 `시세 비교 매물` 섹션에서 별도로 보여준다. 또한 버튼의 숫자는 `marketBasis.sampleCount` 기준이라, 아래 비교 매물 API가 별도로 로드하는 실제 리스트 개수와 어긋날 수 있다.

## Decision

- 예상 순익 카드 안 버튼은 `계산 근거 보기`로 바꾼다.
- 비교 매물 개수와 목록은 바로 아래 `시세 비교 매물` 섹션이 전담한다.
- `계산식 · 비교 매물 N개 보기` 문구는 제거한다.

## Deferred

- `계산 근거 보기` 클릭 시 펼쳐지는 상세 근거의 위치를 예상 순익 카드 아래로 옮길지는 별도 UX 정리에서 판단한다.
