# Wave 309 — 비로그인 프리뷰 예산 힌트 보강

## Context
- 비로그인 메인 프리뷰는 카테고리 힌트를 숨긴 뒤 신뢰/회전 proof chip만 보여주고 있었다.
- 이 구조는 호기심은 만들지만, 사용자가 “내가 살 수 있는 가격대인가?”를 즉시 판단하기 어려워 전환 설득력이 약했다.

## Decision
- 카테고리 힌트는 계속 숨긴다.
- 대신 각 마스킹 매물에 `매입가 10만원 이하`, `매입가 10~30만원` 같은 예산대 chip을 추가한다.
- 정확한 카테고리/모델은 숨기되, 사용자가 예산 적합성은 첫눈에 판단할 수 있게 한다.

## Verification
- `/me` 계약 테스트에 category clue 미노출과 budget chip 회귀 방지를 추가했다.

## Deferred
- 예산 chip 문구가 CTA 전환에 미치는 영향은 실제 랜딩/로그인 전환 로그로 판단한다.
