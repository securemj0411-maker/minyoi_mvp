# 2026-05-22 Wave 528 — 비회원 홈 화면 앱 미리보기형 개편

## 배경

로그인 후 `/me`와 상세 화면은 토스식 앱 톤으로 정리됐지만, 비회원 홈은 예전 마케팅/마스킹 카드 구조가 남아 첫인상이 맞지 않았다.

## 결정

- 비회원 홈은 긴 설명형 랜딩이 아니라 “잠긴 추천 피드 앱 미리보기”처럼 보이게 한다.
- 원색 chip과 과한 뱃지를 줄이고, 흰색/웜그레이/검정 CTA 중심의 차분한 톤으로 맞춘다.
- 원본 매물명과 사진 보안 마스킹은 유지한다.
- “수익 보장”처럼 읽힐 수 있는 문구는 피하고, 같은 상태 비교·비용 반영·상세 리포트 열람 톤으로 설명한다.

## 구현

- `src/components/preview-masked-dashboard.tsx`
  - 상단 safety marquee를 제거하고, hero + locked feed preview 2열 구조로 변경했다.
  - 기존 `LIVE`, 노란/초록 chip, 조건 사진 뱃지를 줄이고 중립적인 상태 라벨로 교체했다.
  - 매물 카드 이미지는 더 큰 rounded 이미지로 보여주되 서버 base64 blur와 CSS blur를 유지한다.
  - CTA는 `로그인하고 보기`, `무료로 시작`으로 단순화했다.
  - 모바일에서는 hero 배지와 혜택 3칸(`무료/1년/0원`)을 숨기고, 제목/설명/CTA 높이를 줄여 첫 화면에 추천 매물 리스트가 바로 보이게 했다.

## 보류

- 비회원 홈에서 “오늘 걸러낸 매물 수”를 토스식 chunk로 다시 보여줄지는 별도 UX 검토 후 결정한다.

## 검증

- `npm run build`
- 모바일 390×844 렌더 확인: 추천 매물 헤더 y≈291, 혜택 3칸은 모바일에서 미노출.
