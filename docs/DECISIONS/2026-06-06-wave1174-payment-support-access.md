# 2026-06-06 Wave 1174 — 결제 중 고객센터 접근

## 결정
- 멤버십 결제/연장 입금 모달 위에서도 1대1 고객센터를 열 수 있게 전역 고객센터 레이어를 결제 모달보다 위로 올렸다.
- 결제 모달 오른쪽 상단에 `문의` 버튼을 추가하고, 같은 고객센터 채팅창을 여는 전역 이벤트(`minyoi:open-support-chat`)로 연결했다.

## 이유
- 결제 모달은 `z-[9990]`이고 기존 고객센터는 `z-[70]/z-[80]`이라 결제 중 고객센터가 뒤에 숨어 있었다.
- 송금/입금확인 단계는 이탈 리스크가 커서, 결제 흐름을 닫지 않고 바로 상담할 수 있어야 한다.

## 보류
- 결제 단계 전용 FAQ/빠른 답변 버튼은 이번 작업에 넣지 않았다. 실제 문의 로그가 쌓인 뒤 자주 묻는 질문만 별도 버튼화한다.
