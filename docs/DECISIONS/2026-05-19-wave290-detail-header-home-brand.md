# 2026-05-19 Wave 290 — 상세 헤더 왼쪽을 홈 브랜드 링크로 변경

## 결정
- `/me` 상품 상세 헤더 왼쪽의 `추천 리포트`/설명 문구를 제거하고 `득템잡이` 브랜드 링크로 바꿨다.
- `득템잡이`를 누르면 `/` 홈으로 이동한다.
- 오른쪽의 `뒤로`/`닫기` 액션은 유지한다.

## 이유
- 모바일 상세는 페이지처럼 동작하므로 왼쪽 상단에 브랜드/홈 진입점이 있는 편이 더 자연스럽다.
- 헤더 설명 문구는 상세 본문과 중복되고 수직 공간을 잡아먹는다.

## 검증
- `tests/me-page-contract.test.ts`에 홈 링크와 `뒤로` 버튼 계약을 함께 추가했다.
