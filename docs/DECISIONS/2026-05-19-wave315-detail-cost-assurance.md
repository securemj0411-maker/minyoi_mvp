# 2026-05-19 Wave 315 — 상품 상세 비용 안심 패널

## 결정
- `/me` 상품 상세 모달 상단에 `최종 매입가 체크` 패널을 추가했다.
- 패널은 상품가, 구매자 배송비 반영 상태, 거래/안전결제 수수료 확인 필요, 재판매 차감 비용을 영수증처럼 보여준다.
- 판매자에게 바로 물어볼 3개 질문을 접힘 영역에 넣고, `문의 문장 복사` 버튼을 추가했다.
- 되돌리기 쉽게 변경 범위는 `src/components/pack-reveal-modal.tsx` UI 컴포넌트 내부로 제한했다.

## 보류
- 상세 API에서 받은 배송 옵션을 모달 state에 저장해 실시간으로 일반/반값 배송비를 재표시하는 작업은 보류했다.
- 번개페이/안전결제 구매자 수수료를 자동 판정하는 parser/DB 필드 추가는 보류했다.
- 사용자별 초보/안전 모드 필터는 별도 UX 실험으로 분리한다.
