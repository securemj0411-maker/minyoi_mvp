# 2026-05-19 Wave 288 — 다른 추천 매물 전환 시 상세 스크롤 초기화

## 결정
- `/me` 상품 상세 안의 "내 다른 추천 매물"을 눌렀을 때 상세 스크롤 영역을 맨 위로 올리도록 했다.
- 모달 전체가 아니라 내부 스크롤 컨테이너에 `ref`를 붙이고, 현재 상세 PID가 바뀔 때 `scrollTo({ top: 0, behavior: "smooth" })`를 실행한다.

## 이유
- 관련 매물 클릭 시 데이터와 URL은 바뀌지만 이전 스크롤 위치가 유지되어 사용자가 페이지가 바뀐 줄 알기 어렵다.
- 매물 전환은 사실상 새 상세 페이지로 이동하는 경험이므로 항상 상단에서 새 상품 사진/제목/차익을 보여줘야 한다.

## 보류
- 전환 토스트나 카드 하이라이트 애니메이션은 보류했다.
- 실제 별도 상세 라우트(`/me/reveals/[pid]`)로 분리하는 작업도 아직 보류한다.
