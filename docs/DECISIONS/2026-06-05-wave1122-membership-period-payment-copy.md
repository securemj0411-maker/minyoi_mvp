# 2026-06-05 Wave 1122 - Membership Period Payment Copy

## Decision

- 신청 플로우의 기간 선택 제목을 `멤버십 기간 선택`으로 명확하게 바꿨다.
- 기간 선택 카드는 선택 상태가 더 강하게 보이도록 프리미엄 톤으로 정리했다.
- `입금했어요` 버튼은 문구를 짧게 유지하고, 5분 자동 가입 안내는 버튼 아래 설명으로 분리했다.
- 입금 후 진행 상태에서는 운영자/수동 확인/시간 경과 표현을 제거하고 `5분 내로 멤버십이 자동 승인됩니다`로 통일했다.

## Deferred

- 기간 선택 화면의 전체 레이아웃 재설계는 보류했다. 이번 작업은 모바일 신뢰감과 문구 혼선을 먼저 줄이는 범위로 제한한다.
