# 2026-05-19 — PG 심사 대비 카피 톤 정비 (Phase 1)

## 결정

비로그인 메인 페이지 + 글로벌 SEO 메타데이터의 카피를 **정보 제공·면책 톤**으로 정비.
"차익 나는 상품", "돈 벌 수 있는 AI 사이트" 같이 **수익 보장·투자 권유로 읽힐 표현 제거**.
브랜드 후크는 "득템(시세 대비 저렴한 매물)" 키워드로 유지.

## 왜 (Why)

PG 가맹 심사(포트원 → 토스페이먼츠/KG/KCP/NICE 채널)에서 미뇨이는 다음 요소 때문에 리스크 분류 가능:

- 외부 중고거래 플랫폼 매물 분석
- AI 차익/시세 비교 정보
- 비실물 디지털 구독
- "돈 벌 수 있음"으로 오해될 여지

심사관이 사이트 첫 화면 + 메타 태그에서 "투자 권유·수익 보장 사이트"로 판단하면 채널 거절 또는 보완 요청 발생.
약관(`/terms` 제6조·제11조)에 면책은 박혀 있으나 **마케팅 카피 자체가 톤다운돼 있어야** 심사 통과 가능성 ↑.

## 안 건드린 것 (의도적)

- **약관 (`/terms`)**: 제6조·제11조에 정보 제공·면책·투자 권유 아님 명시 이미 박혀 있음. 추가 강화 불필요.
- **환불정책 (`/refund-policy`)**: 기존 톤 OK.
- **로그인 후 화면** (`user-reveal-dashboard`, `pack-reveal-modal`, `pack-shop`): "차익" 단어는 정보 제공 맥락이고, `pack-reveal-modal.tsx:2853`·`pack-shop.tsx:180`에 "수익 보장 X" 면책 박혀 있음. PG 심사관이 직접 보긴 어렵고, 봐도 면책 명시되어 OK.
- **`/plans` 가격 페이지**: 카피 점잖음 ("필요한 만큼 추천을 열어보세요"). 그대로 둠.
- **금지 단어** (`보장`·`무조건`·`확정 수익`·`투자`·`재테크`): 사이트 내 사용자 향 카피에서 발견 안 됨.
