# 2026-05-21 — 쉬운모드 상태별 시세 문구 중복 제거

## 결정
- 쉬운모드 비교 매물 단계에서 `같은 상태 매물` 같은 추상 표현을 줄이고, `미개봉품`, `S급 상품`, `사용감 있는 상품`처럼 사용자가 바로 이해하는 상태명으로 말한다.
- 비교 단계 본문은 판단 문장만 담당하고, 아래 상태 기준 카드는 짧은 분류 기준만 보여준다. 같은 `conditionBasisSentence`를 두 번 반복하지 않는다.
- 쉬운모드 비교 매물은 싼 순이 아니라 비싼 순으로 보여준다. 현재 매물이 싸다는 감각을 먼저 주기 위함이다.

## 보류
- 상세 분석 모드의 비교 매물 정렬은 기존 싼 순을 유지했다. 쉬운모드와 상세모드의 목적이 달라서, 전체 정렬 정책은 별도 UX로 다시 본다.
