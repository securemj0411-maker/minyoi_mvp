# Wave 250 — 모바일 상태 칩 복구와 차익 카드 폭 보정

## 결정
- 모바일 추천 카드에서 `일반 ?`, `미개봉/새상품 ?` 같은 상태 등급 칩은 다시 노출한다.
- 상태 칩은 단순 보조 verdict가 아니라, 어떤 등급 기준으로 시세를 계산했는지 설명하는 핵심 신뢰 장치로 본다.
- 차익 카드가 내용 폭만큼만 잡혀 왼쪽으로 치우치는 문제를 막기 위해 본문 영역과 차익 박스에 `flex-1`/`w-full`을 명시했다.

## 이유
- 작은 화면에서도 사용자는 "일반 기준이 뭐냐"를 즉시 확인할 수 있어야 한다.
- 보조 신호는 접어두되, 가격 계산 기준이 되는 상태 등급은 첫 화면에 남기는 편이 신뢰와 가독성 균형이 좋다.
