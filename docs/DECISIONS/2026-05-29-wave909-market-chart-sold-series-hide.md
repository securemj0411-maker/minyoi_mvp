# Wave 909 — 시세 그래프 실거래 표본 0건 처리

## 문제
- 상세보기 시세 그래프에서 판매완료/실거래 표본이 없는데도 `거래가` 범례와 `거래 0건 — 호가 추정` 문구가 노출됐다.
- 사용자 관점에서는 실거래 데이터가 없는데 실거래 차트가 있는 것처럼 보여 신뢰를 떨어뜨린다.

## 결정
- 실거래 표본이 0건이면 거래가 라인, 거래가 점, 거래가 범례, 최근 거래가 문구를 숨긴다.
- 호가 데이터는 구매 판단에 여전히 의미가 있으므로 그래프 자체는 유지하되 제목을 `시세 추이`가 아니라 `호가 추이`로 정직하게 바꾼다.
- 하단 우측 문구도 `거래 0건` 대신 `호가 기준`으로 표시한다.

## 구현
- `src/components/market-history-chart.tsx`
  - `hasSoldHistory`를 추가해 `soldCount > 0`인 sold point가 있을 때만 실거래 series를 그린다.
  - 실거래 표본이 없을 때는 sold 값을 y축 range에도 넣지 않는다.
  - sold 표본이 없으면 title을 `${marketLabel} 호가 N일 추이`로 표시한다.
- `tests/market-history-chart-contract.test.ts`
  - sold 표본 0건일 때 sold series/문구가 숨겨지는 계약 테스트를 추가했다.

## 보류
- 실거래 표본이 충분한 최소 기준을 1건보다 높일지는 추후 운영 데이터 보고 결정한다.
