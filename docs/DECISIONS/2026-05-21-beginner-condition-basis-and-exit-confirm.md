# 2026-05-21 — 쉬운모드 상태별 시세 기준과 원본 이동 전 확인

## 결정
- 쉬운모드 비교 매물 단계에서 `같은 모델`뿐 아니라 `같은 상태` 기준으로 시세를 본다는 점을 명확히 노출한다.
- 원본 매물로 바로 이동시키기 전에 마지막 확인 시트를 한 번 보여준다.
- 마지막 확인 시트는 가격/상태/판매자/구매 전 질문을 짧게 재확인시키고, 사용자가 직접 `확인하고 번개장터 보기`를 누르게 한다.

## 구현
- `conditionBasisSentence`와 `conditionProductLabel`을 추가해 미개봉품, S급, 하자 상품 등 상태별 설명을 쉬운 문장으로 만든다.
- 비교 매물 단계에 `상태까지 맞춰서 시세를 봤어요` 카드를 추가했다.
- 하자가 있는 상품도 깨끗한 상품 시세와 섞지 않고 같은 하자 상품끼리 비교한다는 문구를 추가했다.
- `FixedBunjangFooter`의 원본 보기 CTA를 바로 링크가 아니라 확인 시트를 여는 버튼으로 바꿨다.
- 확인 시트에서 상태별 시세 기준, 판매자 후기 기준, 구매 전 질문을 다시 보여준 뒤 번개장터로 이동한다.

## 보류
- 판매자에게 보낼 질문을 번개장터 채팅으로 직접 전달하는 자동화는 플랫폼 제약과 UX 검토가 필요해 보류한다.
