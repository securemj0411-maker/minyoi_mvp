# 2026-06-05 Wave 1140 - Preserve Feed Signals After Detail Open

## 결정
- 상세 보기 성공 응답의 `data.item`으로 피드 아이템을 통째로 교체하지 않는다.
- 기존 피드 아이템에 상세 응답을 병합하되, 상세 응답에 없는 피드 전용 표시 신호는 유지한다.
- 유지 대상은 당근 거리, 시세 할인 라벨, 평균 회전 라벨, 첫 등록 시점, 거래/셀러 신호 등이다.

## 이유
- 상세 접근 응답은 상세 열람용 최신 데이터라 피드 전용 칩이 빠질 수 있다.
- 기존에는 카드가 `data.item`으로 통째로 바뀌면서 상세 클릭 후 피드 카드 정보가 줄어들었다.

## 보류
- 서버 상세 접근 응답 자체에 모든 피드 표시 신호를 포함시키는 구조 개편은 보류했다.
