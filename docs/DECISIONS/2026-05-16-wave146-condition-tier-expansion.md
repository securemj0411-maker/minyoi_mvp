## 문제

| condition_class | 건수 | 비율 |
|---|---|---|
| **normal** | 1,087 | **65.72%** ← 너무 큼 |
| unopened | 430 | 26.00% |
| worn | 56 | 3.39% |
| clean | 34 | 2.06% |
| mint | 28 | 1.69% |
| flawed | 19 | 1.15% |

**normal 66%** = 매물 description에 condition 표현이 있는데도 parser가 못 잡음 → fallback "normal".

### 25건 normal 매물 분석 결과

실제로는 등급 추출 가능한 표현 다수:
- "실착 1회/3회/5회" → a_grade
- "한 두 번 신음" → a_grade
- "시착만" / "잠깐 신음" → a_grade
- "사이즈 미스/실패/안 맞아" → a_grade (거의 안 신음)
- "보관용" / "보관만 했" → a_grade
- "상태 양호" / "9·10 컨디션" / "EXCELLENT" → b_grade
- "약간의 오염" / "앞코 스크레치" / "미드솔 오염" → c_grade
- "박스채로" / "포장 그대로" / "신어 본 적 없" → s_grade
- "최상급/탑급/특A급" → a_grade

→ 신발 매물 ~60%가 표현 추출 가능. 현재 parser 너무 strict.

## Fix — `parseConditionTier` 정규식 대폭 확장

### S급 추가 (객관적 새상품)
- 박스채로, 박스 그대로, 포장 그대로
- 신은 적 없, 신어본 적 없, 착용해본 적 없
- 풀박 새상품

### A급 추가 (거의 새거)
- 실착 1-3회, 단 1-3회, 구매 후 1-3회
- 한 두 번, 한두번
- 시착만, 시착함
- 사이즈 미스/실패/안 맞/틀
- 보관용, 보관만 했
- 잠깐 (신/착용/밖에 안)
- 셀러 표기: 특A/특S/최상/최상급/탑급 (대소문자 무관)

### B급 추가 (사용감 적음)
- 상태 양호, 상태 좋, 상태 우수, 컨디션 우수
- 9/10, 10/10
- EXCELLENT
- 4-9회 단독 착용 (boundary `(?:^|[^0-9])[4-9]`)
- 깨끗, 깔끔, 관리 잘
- 셀러 표기: 상태 최상

### C급 추가 (사용감 많음)
- 사용감 좀 있, 약간의 오염/얼룩/스크레치
- 앞코 스크레치, 미드솔 오염
- 밑창 닳/마모
- 10+회 착용 (boundary `(?:^|[^0-9])[1-9][0-9]`)
- 발자국

### reject 추가
- 수선 필요, 찌그러짐, 변형 심함, 밑창 벗겨
