# 2026-05-20 Detail Modal Decision Header

## Context

상세 모달은 시세 근거와 동일 제품 비교 매물을 첫 화면 상단에 두어, 사용자가 "이 시세를 믿을 수 있는가"를 바로 판단하게 만드는 흐름이었다.

사용자는 이 위계를 유지하되, 실제 구매 결정을 할 때 필요한 요약 신호가 더 분명하면 좋겠다고 했다.

## Decision

- 기존 비교 매물 위치는 유지한다.
- 예상 순익 카드와 비교 매물 위에 얇은 `PurchaseDecisionHeader`를 추가한다.
- 헤더는 제목/본문/상태 배지만 가진 작은 요약으로 유지한다.
- 모바일 화면 검증 후 중복 칩은 제거했다. 예상 순익 카드가 바로 아래에 같은 숫자와 순익률을 보여주기 때문에, 헤더는 "사도 되나" 판단만 맡긴다.
- 판단 카피는 상태별로 나눈다.
  - 차익 사라짐: 보류
  - 고위험 가품 브랜드: 정품 확인 후 매입 후보
  - 표본/신뢰도 부족: 근거 확인 후 판단
  - 차익과 표본 있음: 근거 있는 매입 후보

## Deferred

- 상세 모달을 별도 상세 페이지로 분리하는 구조 변경은 보류했다.
- Plus CTA와 상세보기 제한 모달의 전체 요금제 문구 재설계는 별도 실험으로 남긴다.
- 실제 사용자 반응을 보고 헤더를 제거하거나 문구를 더 짧게 되돌릴 수 있게 단일 컴포넌트로 유지한다.
