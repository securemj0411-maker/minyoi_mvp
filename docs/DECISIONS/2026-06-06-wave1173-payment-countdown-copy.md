# Wave 1173 — 신규 결제 예약 타이머/입금 안내 문구 정리

## 배경
- 신규 가입 결제 모달에서 `남은 시간`만 보이면 무엇이 남은 시간인지 불명확했다.
- 입금 확인 화면에서 `송금 후 입금했어요 버튼을 누르면 5분 내...` 문구가 헤더와 버튼 아래에 중복 노출됐다.

## 결정
- 신규 가입 예약 타이머 라벨을 `자리 예약 만료까지`로 바꾼다.
- 타이머 카드 안에 `시간 내 입금하지 않으면 취소돼요`를 붙여 7분의 의미를 즉시 이해하게 한다.
- 입금 확인 화면 헤더 설명은 송금/확인 진행 안내만 남기고, 5분 자동 반영 설명은 `입금했어요` 버튼 아래 한 번만 노출한다.

## 보류/주의
- 연장 흐름에는 7분 자리 예약 타이머를 노출하지 않는다.
