# 2026-05-19 Wave 277 — /me 현재 차익 숫자 크기 재조정

## 문제
- Wave 276에서 배경 톤은 낮췄지만, `현재 차익` 숫자 자체가 여전히 너무 크게 보여 카드 안에서 과하게 튀었다.

## 결정
- /me 목록 카드와 상품 모달의 차익 숫자를 `text-base`로 낮췄다.
- 굵기도 `font-bold`에서 `font-semibold`로 낮춰 색으로만 충분히 눈에 띄게 했다.
- 모바일/데스크톱에서 별도 확대(`sm:text-xl`)를 제거해 모달 안에서도 숫자가 과하게 커지지 않게 했다.

## 보류
- 전체 카드 레이아웃 간격 재조정은 보류했다. 이번에는 사용자가 지적한 차익 숫자 크기만 좁게 수정했다.
