# 2026-05-21 — 쉬운모드 인트로 로딩과 셀러 신뢰 스타일 조정

## 결정
- 차단 수치가 늦게 도착해도 인트로 카드 틀은 즉시 보이게 했다.
- 수치 로딩 중에는 "확인 중" 상태를 표시해 화면이 갑자기 비어 보이지 않게 했다.
- "모델 확인 필요" 문구를 "무슨 상품인지 애매"로 바꿨다.
- 판매자 후기/평점 강조는 둥근 pill/card 대신 문장 안 bold와 얇은 구분선 지표로 낮췄다.
- 비교 매물 표본 부족 value note 문구를 제거했다.

## 배경
- 첫 화면 숫자가 늦게 나타나면 사용자에게 계산이 멈춘 것처럼 보인다.
- 판매자 신뢰 카드가 장식적으로 보여 쉬운모드의 토스식 간결함과 맞지 않았다.

## 보류
- 안전 지표 API 응답 속도 자체는 별도 서버/API 캐싱 점검에서 다룬다.
