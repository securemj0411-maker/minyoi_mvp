날짜: 2026-06-06
관련: membership-application-client.tsx 입금 모달, Wave 1195

## owner 지적

입금 단계 버튼:
- "송금 방법 다시 선택" — 거의 안 누름 (보조)
- "입금했어요" — 필수 (모든 사용자가 누름)

둘의 위계가 비슷해서 필수 버튼이 충분히 부각 안 됨.
owner: "입금했어요가 더 크고 가시적이어야. 버튼 안에 '입금 완료 후 이 버튼을 누르면 처리됩니다' 식으로 서브텍스트 넣으면?"

## 변경

"입금했어요" 버튼 (754~):
- 높이 h-12 → **h-14**, 그림자 강화 (`0_14px_30px`)
- **버튼 안 2줄**: "입금했어요"(17px 볼드) + "입금 완료 후 눌러주세요 · 5분 내 자동 반영"(11px, opacity-80)
- 기존 **별도 안내 문단**(idle 상태 "입금 후 입금했어요 버튼을 누르면 5분 내로 자동 반영")을 **버튼 서브텍스트로 흡수** → 정보 분산 제거

"송금 방법 다시 선택" (722~): 그대로 (h-9, text-zinc-400 작은 회색 텍스트 = secondary).

→ 결과 위계: 입금했어요(크고 파란 2줄 CTA) ≫ 송금방법 다시선택(작은 회색 텍스트). 필수 액션이 명백히 dominant.

## 위치(순서)

송금방법 다시선택은 결제수단 안내 박스 하단(수단 변경 맥락)에 유지. 입금했어요는 그 아래 독립 CTA로 크게. 시각 강조로 위계 해소 — 순서 이동(구조 위험)은 하지 않음. 추가로 owner가 순서까지 원하면 후속.
