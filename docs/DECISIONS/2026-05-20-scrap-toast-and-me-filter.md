# 2026-05-20 스크랩 피드백과 /me 빠른 필터

## 결정

- 상세 모달의 스크랩 버튼을 누르면 즉시 짧은 토스트를 보여준다.
- 별도 스크랩 페이지는 만들지 않고, `/me` 상단 카테고리 칩 바의 맨 왼쪽에 `스크랩` 칩을 추가한다.
- `/me`에서 스크랩한 매물은 우선 브라우저 localStorage 스냅샷으로 보관해서, 같은 기기에서 바로 다시 열어볼 수 있게 한다.

## 구현

- `PackRevealModal`에 저장/해제 토스트 상태와 자동 dismiss 타이머를 추가했다.
- `ExploreClient`에 `minyoi_scrap_snapshots_v1` 저장소를 추가하고, 스크랩 칩 활성 시 저장 매물만 표시한다.
- 기존 `minyoi_saved_reveal_pids_v1` PID 저장소도 읽어서 이전 저장 상태가 버튼에 반영되도록 유지했다.

## 보류

- 계정 간 동기화되는 서버 스크랩함은 아직 만들지 않았다. 결제/계정 모델이 더 굳으면 DB 저장으로 승격한다.
- 별도의 스크랩 관리 페이지, 폴더, 메모 기능은 이번 범위에서 제외했다.
