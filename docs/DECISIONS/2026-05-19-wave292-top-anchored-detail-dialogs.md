# 2026-05-19 Wave 292 — 상세 보조 모달 상단 고정

## 배경
- Wave 291에서 위험 신호와 추천 이유 모달을 `top-[39%]` 중심 배치로 올렸지만, 실제 모바일 상세에서는 여전히 사진 아래쪽에 붙어 보이고 “아래에서 튀는” 느낌이 남았다.
- 사용자는 사진을 가려도 좋으니 더 위에서 뜨는 보조 설명 모달을 원했다.

## 결정
- 위험 신호 모달과 추천 이유 모달의 모바일 배치를 중심점 기준에서 상단 기준으로 변경했다.
- 모바일에서는 `top-[72px]`로 띄워 상품 사진 위를 덮는 위치에 표시한다.
- `scale(0.98)` 기반 settle 애니메이션은 제거하고, 짧은 `fade + translateY(-8px)` 애니메이션으로 줄였다.
- 데스크톱은 기존처럼 주변 맥락에서 크게 벗어나지 않도록 별도 위치 규칙을 유지한다.

## 보류
- 공통 Dialog 컴포넌트 추출은 보류한다. 현재는 두 군데만 필요하고, 빠른 UX 보정이 우선이다.
- 로그인된 실제 `/me` 데이터로 시각 QA를 추가하는 것은 별도 세션/계정 준비가 필요해 보류한다.

## 검증
- `/me` 계약 테스트에 상단 고정 위치와 scale 애니메이션 제거 조건을 추가했다.
- 위험 신호 계약 테스트에 같은 위치/모션 조건을 추가했다.
