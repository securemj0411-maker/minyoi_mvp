# 2026-05-18 Wave 230 — /me 안내 카드 숨김과 이모지 톤 다운

## 배경
- `/me`의 피드백 활동 빈 상태와 차단 통계 카드는 반복 방문자에게 학습 비용을 만든다.
- 특히 "아직 신고한 매물이 없어요" 안내와 차단 통계는 한 번 이해하면 매 방문마다 볼 필요가 없다.
- 화면 전반에 장식성 이모지가 많아 제품 UI가 덜 담백해 보인다.

## 결정
- 피드백 활동 빈 상태는 `7일 숨김` 버튼으로 닫을 수 있게 했다.
- 차단 통계 배지와 상단 차단 통계 바도 `숨김` 버튼으로 7일 동안 숨길 수 있게 했다.
- `/me` 주변 주요 장식성 이모지를 inline SVG 아이콘 또는 텍스트로 교체했다.
- "오늘 차익잡이 AI가 차단한 상품 수" 문구는 "오늘 차익잡이가 차단한 매물"로 낮춰 잡았다.

## 보류
- 모든 관리자/디버그 화면의 이모지 제거는 이번 범위에서 제외했다.
- 숨김 상태는 계정 DB가 아니라 브라우저 localStorage 기준이다. 여러 기기 동기화가 필요하면 사용자 설정 테이블로 옮긴다.
