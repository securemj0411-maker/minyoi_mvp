# 2026-05-18 Wave 246 — /me 모바일 첫 CTA 우선순위

## 배경

사용자 지적: 새 유저가 메인페이지의 블러 처리된 추천이 궁금해서 로그인한 뒤 `/me`로 오면, 모바일에서는 `내 피드백 활동` 빈 상태와 보관함 검색/정렬 도구가 먼저 화면을 차지한다. 실제로 기다려야 하는 `가입 환영 매물 준비 중` 안내나 받은 추천 상품 카드가 아래로 밀려 사용자가 받은 줄 모를 수 있다.

## 결정

- `내 피드백 활동`은 `UserRevealDashboard` 아래로 이동한다.
- 피드백 신고가 0건인 빈 상태 위젯은 모바일에서 숨기고, 데스크톱에서만 안내로 유지한다.
- `/me` 모바일 보관함 헤더는 `내 추천 보관함 + 건수 + 판매중/평균` 수준으로 압축한다.
- 검색, 정렬, 카드/목록, 선택/전체삭제는 모바일에서 기본 노출하지 않고 `검색/정렬` 접힘 영역으로 이동한다.
- 신규/로딩 상태에서 추천 상품이 0건이면 검색/정렬 도구를 숨겨 `가입 환영 매물 준비 중` 안내가 바로 보이게 한다.

## 보류

- 카드/목록 모드 자체 제거는 보류한다. 데스크톱과 고사용자용으로 기능은 유지하되 모바일 첫 화면에서는 숨긴다.
- 피드백 활동의 노출 시점을 가입 후 1시간 같은 시간 기준으로 제어하는 것은 보류한다. 우선 `신고 0건 모바일 숨김`과 위치 이동으로 해결한다.

## 검증

- `/me` contract test에 상품 보관함이 피드백 활동보다 먼저 나오는지, 모바일 빈 피드백 위젯이 숨겨지는지, 검색/정렬 도구가 접힘 영역으로 들어갔는지 확인하는 계약을 추가했다.
