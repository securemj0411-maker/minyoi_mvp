# 2026-06-04 Wave 1093 - Feed Offer Server Expiry

## 결정

- 피드 멤버 전용 1시간 이벤트 노출 타이머에서 로컬스토리지 의존을 제거했다.
- `/api/membership/status`가 최신 승인 신청의 활성화 시각과 서버 기준 `memberOfferExpiresAt`을 내려준다.
- 피드 이벤트 배너는 서버가 내려준 `memberOfferExpiresAt` 기준 남은 시간이 0보다 클 때만 렌더링한다.

## 이유

- 로컬스토리지 타이머는 새 브라우저, 스토리지 초기화, 기기 변경, 배포/재접속 상황에서 이벤트 상태가 다시 살아날 수 있다.
- 결제/업셀 이벤트의 유효성은 사용자별 DB 상태 기준이어야 한다.

## 보류

- 이벤트 클릭/노출 이력 테이블은 아직 추가하지 않았다. 현재는 승인 시각 기준 1시간 만료만 서버에서 강제한다.
