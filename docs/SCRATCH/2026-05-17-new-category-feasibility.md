# 새 카테고리 진입 가능성 조사 (2026-05-17)

사용자 명령: "3개 다 조사해보고 결정 가능성 보자" — DJI 드론/GoPro / 헤어 기기 / 향수.

신발급 기준:
- 매물 supply 큼
- 일반인 친화
- 모델 명확 (시세 비교 가능)
- 짝퉁 risk 작음 또는 검증 가능
- 차익 매물 일정 비율

## 1. DJI 드론 + 액션캠 + GoPro 🥇

- 한국 드론/액션캠 시장 활발 (특히 DJI Mini 시리즈 인기 폭발)
- 차익 매물 큼 — 신모델 출시 시 구모델 매물 쏟아짐

**DJI 액션캠**:
| 모델 | 출시 |

### 옵션
- **드론**: 본체 only vs Fly More Combo (배터리 3개 + 컨트롤러 + 케이스 포함, +10~30만)
- **GoPro/Osmo Pocket**: 단일 옵션 (메모리는 별매)

### Risk 평가
| 항목 | 평가 |
|---|---|
| 매물 supply | 매우 활발 (신발급) |
| 모델 명확성 | ✓ Apple 수준 명확 |
| 가격대 | 30~200만 (일반인 친화) |
| 짝퉁 risk | **거의 없음** (정품 등록 + 시리얼) |
| 운영 risk | 작음 (부품 분해 매물 mustNotContain) |
| 차익 효과 | **큼** (신모델 사이클 6~12개월) |

### 작업량
- catalog SKU: 15~20개
- parser: 드론 컴보 (Fly More Combo) vs 본체 only 인식 추가
- mining query: 15+ query
- **예상 commit 1~2개**

### 결론
**🥇 가장 추천.** 짝퉁 risk 거의 없음 + 매물 활발 + 모델 명확 + 작업량 적정.

## 2. 헤어 기기 (다이슨 / 시아루스 / 파나소닉) 🥈

### 시장
- 다이슨 슈퍼소닉/에어랩 한국 인기 폭발 (특히 에어랩)
- 시아루스 글램팜/매직 활발 (저렴한 대안)
- 파나소닉 나노이 활발

### 라인업

**Dyson**:
| 모델 | 출시 | 가격 |
|---|---|---|
| Dyson Supersonic (HD08) | 2016, 한국 출시 2018+ | 50~60만 |
| Dyson Supersonic Origin | 2023 | 40만 |
| Dyson Airwrap (HS01 → HS05 i.d.) | 2018~2024 | 60~80만 |
| Dyson Airwrap Co-anda 2x | 2024 | 90만 |
| Dyson Corrale (HS07) | 2020 | 60만 |

**시아루스**:
- 시아루스 글램팜 (여러 시즌)
- 시아루스 매직 ProV
- 시아루스 매직 인피니티

**파나소닉**:
- EH-NA0J (나노이 + 미네랄)
- EH-NA9C
- EH-NA98

**바비리스 / 차홈** 등 추가 가능

### 옵션
- 거의 단일 옵션 (색상만 변형, 시세 동일)
- Airwrap 변형: i.d. vs 옛 모델 vs Co-anda 2x — 분리 필요

### Risk 평가
| 항목 | 평가 |
|---|---|
| 매물 supply | 활발 (저렴한 가격대) |
| 모델 명확성 | ✓ |
| 가격대 | 20~90만 (저렴) |
| 짝퉁 risk | **다이슨 일부 있음** (박스/시리얼로 식별 가능). 시아루스/파나소닉 작음 |
| 운영 risk | 작음 |
| 차익 효과 | 중간 (낮은 가격대 → 차익 매물 비율은 작음) |

### 작업량
- catalog: 5~10개
- parser: 단순 (옵션 거의 없음)
- mining query: 5~10개
- **예상 commit 1개**

### 결론
**🥈 작업 빠름 + 안전.** 단 차익 매물 비율 작음 (저렴한 가격대).

## 3. 향수 🥉

### 시장
- 한국 명품/럭셔리 향수 시장 활발
- 매물 supply 매우 큼 (소규모 거래)

### 라인업 (브랜드별)

**조말론 (Jo Malone)**: 우드세이지 / 라임바질 / 잉글리쉬페어 / 블랙베리베이 / 우드세이지 / 피오니블러쉬 등 30+ 향

**르라보 (Le Labo)**: 산탈33 / 더 누아 29 / 베르가못 22 / 로즈 31 / 일랑49 등 20+ 향

**딥디크 (Diptyque)**: 필로시코스 / 도손 / 오 카피탈 / 탐다오 / 오 듀엘 등 30+ 향

**톰포드 (Tom Ford)**: 블랙오키드 / 토바코바닐라 / 로스트체리 / 우드앤레더 등 50+ 향

**메종 마르지엘라 (Replica)**: 재즈클럽 / 바이더파이어플레이스 / 비치워크 등 20+ 향

**기타**: 메모 (Memo) / 펜할리곤스 / 이그자드 / 산타마리아 노벨라 등

### 옵션
- **용량**: 10ml / 30ml / 50ml / 100ml / 200ml — 용량별 시세 크게 다름 (필수 분리)
- **사용량**: 100% (미개봉) / 90%+ / 70%+ / 50%+ — 매물 시세 영향 큼

### Risk 평가
| 항목 | 평가 |
|---|---|
| 매물 supply | **매우 활발** |
| 모델 명확성 | 향수명 명확. 단 라인업 수백 종 |
| 가격대 | 5~30만 (저렴) |
| 짝퉁 risk | **일부 있음** (병 모양/뚜껑/박스 식별 가능. 명품 가방보다 낮음). 분주/소분 매물도 있음 |
| 운영 risk | 사용량 명시 매물 (분리 시세) + 분주 매물 + 빈병 매물 noise |
| 차익 효과 | 중~큼 (인기 향 한정판/단종 시세 폭등) |

### 작업량 (매우 큼)
- catalog SKU: 100+ 가능 (브랜드 × 향 × 용량 조합)
- 또는 broad SKU (브랜드 + 인기 향만): 30~50개
- parser: 용량 (ml) 추출 + 사용량 (50%/100% 미개봉) 인식
- mining query: 50+ query
- **예상 commit 3~5개**

### 결론
**🥉 큰 효과 but 큰 작업.** 라인업 폭발적 + 옵션 (용량/사용량) 복잡. 신중히 진행 필요.
